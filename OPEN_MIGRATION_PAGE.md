## 🎯 ماذا يفعل Migration؟

```
يقرأ كل سيارة في Firestore:
┌────────────────────────┐
│ city: 'София'          │ ← قديم
└────────────────────────┘
         ↓
يحوّلها إلى:
┌────────────────────────┐
│ location: {            │ ← جديد
│   cityId: 'sofia-city',│
│   cityNameBg: 'София', │
│   coordinates: {...}   │
│ }                      │
└────────────────────────┘
         ↓
يحفظها في Firestore ✅
         ↓
الخريطة تجدها الآن! ✅
```
