## الميزات الجديدة المضافة

### 1. نظام الأدوار (RBAC)
**كل مستخدم الآن لديه دور محدد:**
- **Buyer** (المشتري) - الدور الافتراضي
- **Seller** (البائع) - يمكن إنشاء إعلانات سيارات
- **Admin** (المدير) - صلاحيات كاملة

### 2. نظام المراسلات
**يمكنك الآن:**
- إرسال رسائل مباشرة لأي مستخدم
- استقبال إشعارات فورية عند وصول رسالة
- رؤية المحادثات القديمة
- معرفة عدد الرسائل غير المقروءة

### 3. نظام التقييمات
**يمكن للمشترين:**
- تقييم البائعين (1-5 نجوم)
- كتابة مراجعات نصية
- رؤية تقييمات البائعين الآخرين

### 4. لوحة تحكم البائعين
**للبائعين فقط:**
- عرض إحصائيات شاملة
- متابعة المشاهدات والاستفسارات
- معرفة معدل التحويل
- رؤية السيارة الأكثر مشاهدة

## كيفية الاستخدام

### للمشترين:

#### 1. عرض بروفايل بائع:
```
الخطوات:
1. اذهب إلى http://localhost:3000/users
2. اختر أي بائع
3. سترى:
   - معلومات البائع
   - تقييماته
   - سياراته المعروضة
   - زر "Send Message"
```

#### 2. إرسال رسالة:
```
الخطوات:
1. اضغط على زر "Send Message" في بروفايل البائع
2. سيتم توجيهك إلى صفحة الرسائل
3. اكتب رسالتك واضغط "Send"
4. ستستلم إشعار عند الرد
```

#### 3. كتابة مراجعة:
```
الخطوات:
1. اذهب إلى صفحة السيارة
2. انزل إلى قسم "Seller Reviews"
3. اضغط "Write a Review"
4. اختر التقييم (1-5 نجوم)
5. اكتب تعليقك
6. اضغط "Submit Review"
```

### للبائعين:

#### 1. الترقية إلى بائع:
```
الخطوات:
1. اذهب إلى http://localhost:3000/profile
2. اضغط "Upgrade to Seller" (إذا ظهر)
3. أدخل معلومات الأعمال (اختياري):
   - اسم الشركة
   - رقم BULSTAT
   - العنوان
4. اقبل الشروط والأحكام
5. اضغط "Upgrade"
6. الآن يمكنك إنشاء إعلانات!
```

#### 2. عرض لوحة التحكم:
```
الخطوات:
1. اذهب إلى http://localhost:3000/profile?tab=analytics
2. سترى:
   - عدد السيارات (مجموع/نشطة/مباعة)
   - المشاهدات والاستفسارات
   - متوسط التقييم
   - معدل التحويل
   - النشاط الأخير
```

#### 3. إدارة السيارات:
```
الخطوات:
1. اذهب إلى http://localhost:3000/profile?tab=myads
2. سترى جميع سياراتك
3. يمكنك:
   - تحرير
   - حذف
   - عرض الإحصائيات
```

## النشر للإنتاج

### 1. نشر Cloud Functions:
```bash
cd functions
npm install
npm run build
firebase deploy --only functions
```

### 2. نشر Firestore Rules:
```bash
firebase deploy --only firestore:rules
```

### 3. نشر Frontend:
```bash
cd bulgarian-car-marketplace
npm install
npm run build
firebase deploy --only hosting
```

## الصفحات الجديدة

### صفحة الرسائل:
**URL:** `http://localhost:3000/messages`

**الميزات:**
- قائمة جميع المحادثات
- نافذة دردشة فورية
- عدادات الرسائل غير المقروءة
- تصميم responsive

### صفحة دليل المستخدمين (محدّثة):
**URL:** `http://localhost:3000/users`

**التحديثات:**
- عرض تقييم كل بائع
- زر "Send Message" في كل بطاقة
- فلاتر إضافية
