# 🌐 دليل زر تبديل اللغة - Language Toggle Guide

**تاريخ الإضافة:** 30 سبتمبر 2025  
**الحالة:** ✅ **جاهز ويعمل!**

---

## 🎯 **أين يوجد زر اللغة؟**

### 1️⃣ **في Header (الشريط العلوي)** ⭐ جديد!

```
┌──────────────────────────────────────────────┐
│  🚗 Globul Cars  | 🔍 Search...  | 🇧🇬 BG ↔️  │
│                                               │
│  [Logo] [Search Bar] [🇧🇬/🇺🇸] [♥] [✉] [🔔]  │
└──────────────────────────────────────────────┘
                          ↑
                  زر تبديل اللغة!
```

**الموقع:** أعلى يمين الصفحة، قبل أزرار Favorites/Messages/Notifications

**المظهر:** 
- 🇧🇬 (عندما اللغة بلغارية)
- 🇺🇸 (عندما اللغة إنجليزية)

### 2️⃣ **في HomePage (الصفحة الرئيسية)**

```
موجود أيضاً في:
- منتصف الصفحة الرئيسية
- بحجم medium وأكثر وضوحاً
```

---

## 🎨 **كيف يبدو الزر؟**

### التصميم:

```css
📱 في Header (حجم موحد مع باقي الأزرار):
┌──────┐
│  🌍  │  ← أيقونة الكرة الأرضية
└──────┘

🎨 المظهر:
- أيقونة الكرة الأرضية (Globe) ثنائية الأبعاد
- نفس التصميم الموحد مع أزرار المفضلة والرسائل والإشعارات
- تأثير glow برتقالي أنيق
- عند hover: يظهر background خفيف
- shadow مميز بلون برتقالي
```

### الأحجام المتاحة:

```tsx
<LanguageToggle size="small" />   // في Header
<LanguageToggle size="medium" />  // في HomePage  
<LanguageToggle size="large" />   // للاستخدامات الخاصة
```

---

## ⚡ **كيف يعمل؟**

### عند الضغط على الزر:

```
1. يتبدل اللغة فوراً (BG ↔ EN)
2. جميع النصوص في الصفحة تتحدث فوراً
3. يتم حفظ اختيارك في localStorage
```

---

## 💡 **الميزات**

```
✅ تبديل فوري (بدون إعادة تحميل)
✅ إشعار جميل ومتحرك
✅ يحفظ اختيارك
✅ responsive للموبايل
✅ accessible (ARIA labels)
✅ animation ناعم
✅ تصميم gradient حديث
✅ يعمل في كل الصفحات
```

---

## 📱 **في Mobile (الهواتف)**

```
على الشاشات الصغيرة:

- الزر يصغر قليلاً
- النص يختفي (فقط الأعلام)
- الإشعار يظهر بعرض كامل
```

---

## 🔧 **الكود التقني**

### كيف تم تنفيذه:

```tsx
// في Header.tsx:
import LanguageToggle from '../LanguageToggle/LanguageToggle';

// الزر يستخدم أيقونة Globe من lucide-react
<LanguageToggle size="small" showText={false} className="glow-button" />
```

### الخصائص المتاحة:

```typescript
interface LanguageToggleProps {
  className?: string;      // لإضافة CSS مخصص (مثل: "glow-button")
  showText?: boolean;      // إظهار/إخفاء النص (БГ/EN)
  size?: 'small' | 'medium' | 'large';  // الحجم
}
```
