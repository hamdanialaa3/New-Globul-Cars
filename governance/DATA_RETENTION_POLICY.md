# Data Retention & Deletion Policy\n## Overview\nKoli One complies with GDPR, CCPA, and regional data protection laws. This policy defines how long we retain data and how users exercise their right to be forgotten.\n\n## Data Retention Schedule\n\n| Data Type | Retention | Purpose | Deletion Method |\n|-----------|-----------|---------|------------------|\n| **User Accounts** | | | |\n| Profile (name, email, phone) | Until deletion requested | Identity & contact | Anonymize (hash PII) |\n| Auth tokens | 30 days (after logout) | Session | Auto-expire |\n| Login audit log | 90 days | Security | Auto-delete |\n| **Listings & Content** | | | |\n| Published listing | 3 years (from last activity) | Historical record | Hard delete |\n| Draft listing | 30 days (inactive) | Workspace cleanup | Auto-delete |\n| Images/media | Until listing deleted | Content reference | Cascade delete |\n| **Transactions** | | | |\n| Payment records | 7 years | Tax/audit | Archive → delete |\n| Invoice records | 7 years | Accounting | Archive → delete |\n| Refund history | 3 years | Dispute resolution | Anonymize |\n| **Communications** | | | |\n| Direct messages | 2 years (after last interaction) | Reduced disk usage | Archive after 2y, delete after 3y |\n| Support tickets | 2 years | Support history | Archive after 2y |\n| Notifications | 30 days | Transient | Auto-delete |\n| **Analytics & Logs** | | | |\n| Page view logs | 90 days | Performance analysis | Auto-delete |\n| Error logs (Sentry) | 30 days | Debugging | Auto-delete |\n| Audit logs (security) | 3 years | Compliance | Archive |\n| **Consent Records** | | | |\n| Opt-in records (marketing) | Until opt-out | Marketing preferences | Hard delete on opt-out |\n| Consent timestamps | Duration of consent | Legal proof | Archive |\n| **ML Training Data** | | | |\n| Images used for training | Per consent (default 1 year) | Model accuracy | Delete if opt-out requested |\n| Salary/vehicle data | 180 days (aggregated) | Market analytics | Aggregate → anonymize |\n\n## User Rights: Right to be Forgotten\n\n### Scope\nUser can request deletion of:\n- Profile & personal data\n- All listings (including historical)\n- Messages (all conversations)\n- Transactions (replaced with anonymized records for tax purposes)\n- Analytics & audit logs (except those legally required)\n\n### Exclusions (Cannot Delete)\n- Tax records (7-year legal requirement)\n- Fraudulent activity logs (investigation purposes)\n- Court-ordered records\n\n### Process\n\n1. **User initiates request**\n   - Web form: Account Settings → Delete Account → Reason\n   - Email: privacy@koli.one with subject \"GDPR Deletion Request\"\n   - Mobile: Settings → Privacy → Request Data Deletion\n\n2. **Verification (24 hours)**\n   - Email confirmation sent to user\n   - User clicks link to confirm\n   - Admin notified in #privacy-requests Slack\n\n3. **Execution (14 days)**\n   - All profile data anonymized\n   - All listings archived (inaccessible to public)\n   - All messages deleted (not recoverable)\n   - Payment records replaced with anonymized invoices\n   - Confirmation email sent\n\n4. **Verification (30 days)**\n   - Compliance team audits deletion\n   - Check backups for residual PI\n   - Sign off in deletion log\n\n### Timeline\n- User request: T+0\n- Verification deadline: T+1 day\n- Deletion deadline: T+14 days\n- Backup cleanup: T+30 days (standard backup retention)\n- Final audit: T+45 days\n\n## Data Export (Subject Access Request)\n\n### Available\n- Profile information\n- All listings (published & draft)\n- Messages (encrypted, last 2 years)\n- Transaction history\n- Analytics (anonymized)\n\n### Excluded\n- System logs\n- Internal notes\n- Fraud detection flags\n\n### Process\n1. User requests via Account Settings → Privacy → Download My Data\n2. System generates ZIP file containing:\n   - User profile JSON\n   - Listings CSV\n   - Messages JSON\n   - Transactions CSV\n3. Email with download link (24-hour expiry)\n4. User downloads (logged)\n\n## Deletion Verification Checklist\n\n- [ ] Firestore documents deleted\n- [ ] Cloud Storage files deleted\n- [ ] Algolia index entries removed\n- [ ] Backups tagged with retention policy\n- [ ] Vault secrets deleted\n- [ ] Third-party integrations notified (if applicable)\n- [ ] Email confirmed deletion\n\n## Contact\n\n- **Privacy/GDPR**: privacy@koli.one\n- **Data Operations**: ops-team@koli.one\n- **Legal**: legal@koli.one\n\n## References\n\n- GDPR Article 17 (Right to be Forgotten)\n- GDPR Article 20 (Data Portability)\n- Privacy policy: https://koli.one/privacy\n"
