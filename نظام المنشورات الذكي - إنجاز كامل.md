## الميزات الكاملة

### Smart Algorithm (الخوارزمية الذكية)

**Scoring System (100 points):**
```
40 points: Engagement
  - Reactions: ×1
  - Comments: ×3
  - Shares: ×5
  - Saves: ×4
  - Views: ×0.01

20 points: Recency
  - < 1 hour: 20 points
  - < 3 hours: 18 points
  - < 6 hours: 15 points
  - < 12 hours: 12 points
  - < 24 hours: 8 points
  - < 72 hours: 4 points
  - Older: 1 point

20 points: Quality
  - Has images: +5
  - Has video: +7
  - Car reference: +5
  - Long text (>200): +3
  - Has hashtags: +2
  - No spam reports: +3

20 points: Author
  - Trust score: 0-10
  - Followers: 0-5
  - Verified: +5
  - Profile type: +2-3
```

**Personalization (+26 bonus):**
```
+10: Favorite car brands
+5: Favorite authors
+3: Favorite locations
+5: Favorite hashtags
+3: Favorite post types
```

**ML Algorithms:**
```
Collaborative: Similar users' likes
Content-Based: Similar content matching
Hybrid: 40% + 30% + 20% + 10%
```

**Feed Sources:**
```
50%: Following (users you follow)
30%: Trending (high engagement)
20%: Local (your city)
```

**Diversity Filter:**
```
- No same author 2x in a row
- Max 2 same type consecutively
- Reset every 5 posts
```
