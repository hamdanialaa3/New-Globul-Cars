## 🎯 What is This Folder?

This folder (`project-tree/`) contains **comprehensive documentation** for all major sections and features of the Koli One (Коли-Уан) Bulgarian car marketplace application.

**Purpose:** Provide detailed, professional documentation that explains:
- What each system does
- How it works
- Technical implementation details
- Code examples
- Best practices

**Target Audience:**
- Developers working on the project
- New team members
- AI assistants (like Cursor AI)
- Future maintainers
