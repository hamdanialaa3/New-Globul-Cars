## ✅ قائمة الاختبار - نظام المفضلة

### 🧪 اختبارات يدوية

#### أ. اختبار زر القلب في الصفحة الرئيسية (`/cars`)

- [ ] **الزر يظهر**: تحقق من وجود ❤️ في الزاوية العلوية اليمنى من كل بطاقة
- [ ] **اللون الأول**: القلب يجب أن يكون رمادي أولاً (بدون تعبئة)
- [ ] **تأثير عند المرور**: يتغير اللون عند المرور
- [ ] **النقر الأول**: يصبح أحمر ممتلئ
  - ✅ يظهر Toast: "❤️ Added to favorites!"
  - ✅ البيانات تُحفظ في Firestore
- [ ] **النقر الثاني**: يعود للرمادي
  - ✅ يظهر Toast: "Removed from favorites"
  - ✅ يتم الحذف من Firestore

#### ب. اختبار صفحة المفضلات (`/favorites`)

**الوصول والتحميل:**
- [ ] يمكن الوصول إلى `/favorites` مباشرة
- [ ] تظهر الصفحة بسرعة
- [ ] الرسالة الفارغة تظهر إذا لم تكن هناك مفضلات

**العرض والتصميم:**
- [ ] الخلفية الداكنة مع الشبكة تظهر بشكل صحيح
- [ ] البطاقات لها Glass-morphism effect
- [ ] الصور تحميل بشكل صحيح
- [ ] العنوان يعرض العدد الصحيح

**الفلاتر:**
- [ ] **ترتيب** (Sort By):
  - [ ] "Newest First" يرتب حسب التاريخ الأحدث
  - [ ] "Oldest First" يرتب عكسياً
  - [ ] "Price: Low to High" يرتب من الأقل للأعلى
  - [ ] "Price: High to Low" يرتب من الأعلى للأقل
  - [ ] "Car Name A-Z" يرتب أبجدياً

- [ ] **نطاق السعر** (Price Range):
  - [ ] "All Prices" يعرض الكل
  - [ ] "Under €10,000" يعرض فقط <10k
  - [ ] "€10,000 - €20,000" يعرض النطاق الصحيح
  - [ ] "€20,000 - €50,000" يعرض النطاق الصحيح
  - [ ] "Over €50,000" يعرض فقط >50k

- [ ] **نوع الوقود** (Fuel Type):
  - [ ] "All Fuel Types" يعرض الكل
  - [ ] "Petrol" يعرض فقط البنزين
  - [ ] "Diesel" يعرض فقط الديزل
  - [ ] "Electric" يعرض فقط الكهربائية
  - [ ] "Hybrid" يعرض فقط الهجينة

- [ ] **التروس** (Transmission):
  - [ ] "All Types" يعرض الكل
  - [ ] "Manual" يعرض يدوي فقط
  - [ ] "Automatic" يعرض أوتوماتيكي فقط

**العروض المختلفة:**
- [ ] Grid view (4-5 أعمدة بناءً على الحجم)
- [ ] List view (صفحة واحدة بطول طويل)
- [ ] التبديل بينهما يعمل فوراً

**بيانات البطاقات:**
- [ ] صورة السيارة
- [ ] اسم (Make Model)
- [ ] السعر الحالي
- [ ] السعر الأصلي مشطوب (إن وجد)

- [ ] السنة
- [ ] المسافة المقطوعة
- [ ] الموقع
- [ ] نوع الوقود

**الأزرار:**
- [ ] زر "View" أزرق - ينقل إلى `/car/:id`
- [ ] زر "Remove" أحمر - يحذف من المفضلات
