3. script.js (الوظيفة)
هذا الملف يضيف وظيفة التبديل (toggle) للزر عند النقر عليه.

JavaScript

document.addEventListener('DOMContentLoaded', () => {
    const switchContainer = document.querySelector('.switch-container');

    // إضافة مستمع لحدث النقر على الزر بالكامل
    switchContainer.addEventListener('click', () => {
        // تبديل الفئة (class) 'on' لإضافة/إزالة التأثيرات
        switchContainer.classList.toggle('on');
    });
});
classList.toggle('on'): هذه الوظيفة هي السر. فهي تضيف الفئة (class) المسماة on عند النقر الأول، وتزيلها عند النقر الثاني، وهكذا. هذا يسمح لنا بالتحكم في كل تأثيرات CSS باستخدام فئة واحدة فقط.

## 📝 المرحلة 2: المتطلبات القانونية في بلغاريا

### 2.1 متطلبات Car Dealership / Trader
**حسب القانون البلغاري:**

#### معلومات إلزامية:
1. **رقم الضريبة (ДДС номер / VAT Number)**
   - Format: BG + 9 أو 10 أرقام
   - مثال: BG123456789
   - إلزامي للشركات المسجلة

2. **اسم التجاري (Търговско наименование / Trade Name)**
   - الاسم الرسمي المسجل
   - يجب أن يطابق السجل التجاري

3. **العنوان التجاري (Търговски адрес / Business Address)**
   - العنوان الكامل مع المدينة والرمز البريدي
   - يجب أن يكون عنوان فعلي وليس صندوق بريد

4. **رقم التسجيل التجاري (ЕИК / UIC - Unified Identification Code)**
   - رقم تعريفي فريد من 9 أرقام
   - مثال: 123456789

5. **معلومات الاتصال التجاري:**
   - هاتف العمل
   - البريد الإلكتروني التجاري
   - ساعات العمل

6. **وصف النشاط التجاري**
   - نوع التجارة (سيارات جديدة/مستعملة/كليهما)
   - الخدمات المقدمة

### 2.2 متطلبات Corporate Company
**للشركات الكبرى:**

#### معلومات إلزامية إضافية:
1. **رقم الشركة الموحد (БУЛСТАТ / BULSTAT)**
   - 9 أرقام للشركات
   - مثال: 123456789

2. **الشكل القانوني للشركة**
   - ООД (OOD - Limited Liability Company)
   - ЕООД (EOOD - Single-Member LLC)
   - АД (AD - Joint Stock Company)

3. **رأس المال المسجل**
   - المبلغ الرسمي المسجل

4. **الإدارة التنفيذية**
   - أسماء المديرين التنفيذيين (اختياري)

5. **معلومات الشركة**
   - سنة التأسيس
   - عدد الموظفين (اختياري)
   - فروع الشركة (إن وجدت)

### 3.2 Firestore Security Rules

```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId} {
      allow read: if request.auth != null;
      allow write: if request.auth != null && request.auth.uid == userId;
      
      // التحقق من صحة بيانات Dealer
      allow update: if request.auth.uid == userId &&
        (request.resource.data.profileType != 'dealer' ||
         (request.resource.data.dealerInfo.vatNumber.matches('^BG[0-9]{9,10}$') &&
          request.resource.data.dealerInfo.uicNumber.matches('^[0-9]{9}$')));
      
      // التحقق من صحة بيانات Company
      allow update: if request.auth.uid == userId &&
        (request.resource.data.profileType != 'company' ||
         (request.resource.data.companyInfo.vatNumber.matches('^BG[0-9]{9,10}$') &&
          request.resource.data.companyInfo.bulstatNumber.matches('^[0-9]{9}$')));
    }
  }
}
```

### 8.2 نظام الانتقالات السلسة (Smooth Transitions)

#### التحسين: انتقالات متدرجة عند تغيير الثيم

**الملف:** `src/contexts/ProfileTypeContext.tsx` - تحديث

```typescript
// إضافة transition state
const [isTransitioning, setIsTransitioning] = useState(false);
const [transitionProgress, setTransitionProgress] = useState(0);

const switchProfileType = async (newType: ProfileType) => {
  setIsTransitioning(true);
  
  // Smooth color transition animation
  const duration = 1000; // 1 second
  const steps = 60; // 60 fps
  const stepDuration = duration / steps;
  
  for (let i = 0; i <= steps; i++) {
    await new Promise(resolve => setTimeout(resolve, stepDuration));
    setTransitionProgress(i / steps);
  }
  
  // Update the actual theme
  setProfileType(newType);
  
  // Reset
  setIsTransitioning(false);
  setTransitionProgress(0);
};

// Interpolate colors during transition
const getTransitionColor = (fromColor: string, toColor: string, progress: number) => {
  // RGB interpolation logic
  const from = hexToRgb(fromColor);
  const to = hexToRgb(toColor);
  
  const r = Math.round(from.r + (to.r - from.r) * progress);
  const g = Math.round(from.g + (to.g - from.g) * progress);
  const b = Math.round(from.b + (to.b - from.b) * progress);
  
  return `rgb(${r}, ${g}, ${b})`;
};
```

const SparklineChart: React.FC<{ data: number[] }> = ({ data }) => {
  const max = Math.max(...data);
  const points = data.map((value, index) => {
    const x = (index / (data.length - 1)) * 100;
    const y = 100 - (value / max) * 100;
    return `${x},${y}`;
  }).join(' ');
  
  return (
    <svg width="100" height="30" style={{ marginTop: '8px' }}>
      <polyline
        points={points}
        fill="none"
        stroke="currentColor"
        strokeWidth="2"
        strokeLinecap="round"
        strokeLinejoin="round"
      />
    </svg>
  );
};

```typescript
const useKeyboardShortcuts = () => {
  useEffect(() => {
    const handleKeyPress = (event: KeyboardEvent) => {
      // Ctrl/Cmd + E: Edit Profile
      if ((event.ctrlKey || event.metaKey) && event.key === 'e') {
        event.preventDefault();
        navigateToEdit();
      }
      
      // Ctrl/Cmd + N: New Listing
      if ((event.ctrlKey || event.metaKey) && event.key === 'n') {
        event.preventDefault();
        createNewListing();
      }
      
      // Ctrl/Cmd + P: Toggle Privacy
      if ((event.ctrlKey || event.metaKey) && event.key === 'p') {
        event.preventDefault();
        togglePrivacy();
      }
      
      // Escape: Close modals
      if (event.key === 'Escape') {
        closeAllModals();
      }
    };
    
    window.addEventListener('keydown', handleKeyPress);
    return () => window.removeEventListener('keydown', handleKeyPress);
  }, []);
};
```
