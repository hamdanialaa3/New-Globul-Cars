# Contributing to Koli.one

Thank you for your interest in Koli.one — Bulgaria's Leading Car Marketplace!

## 🚗 About the Project

Koli.one is a proprietary multi-vendor car marketplace platform serving Bulgaria.
Built with React 18 + TypeScript + Firebase, it features AI-powered car valuation,
smart search, and dealer tools.

## 📋 How to Contribute

This is a proprietary project. External contributions are welcome for:

- 🐛 **Bug Reports**: Open an issue describing the bug
- 💡 **Feature Requests**: Open an issue with your idea
- 📝 **Documentation**: Improvements to docs and guides
- 🌍 **Translations**: Bulgarian, English, Arabic language improvements

## 🔧 Development Setup

```bash
git clone https://github.com/hamdanialaa3/New-Globul-Cars.git
cd New-Globul-Cars

# Web app (run inside the web/ folder)
cd web
npm ci
cp .env.example .env.local
npm start          # or: npm run start:dev

# Mobile app (run inside the mobile_new/ folder)
cd ../mobile_new
npm ci
npm start          # expo start
```
