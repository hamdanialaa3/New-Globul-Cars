### 3. **التحقق غير الكافي في ProfileTypeContext** 🔴 **CRITICAL**

#### المشكلة:

```typescript
// في ProfileTypeContext.switchProfileType()
if (newType === 'dealer') {
  if (userData.profileType === 'dealer' && userData.dealershipRef) {
    // التحقق فقط إذا كان بالفعل dealer
  } else if (newType === 'dealer') {
    // ❌ يطلب dealershipRef لكن لا يوجد مكان لإنشائه!
    throw new Error('Cannot switch to dealer profile: Missing dealershipRef.');
  }
}
```

#### المشكلة:
- ❌ المستخدم يريد التحويل إلى Dealer لأول مرة
- ❌ النظام يطلب `dealershipRef` الذي لا يوجد
- ❌ لا يوجد توجيه لصفحة التسجيل

#### الحل:
- التحقق من وجود بيانات التسجيل بدلاً من `dealershipRef`
- توجيه المستخدم لصفحة التسجيل إذا لم يكمل البيانات
