# نظام البحث الجديد للسيارات - دليل الاستخراج الشامل

## نظرة عامة
تم تطوير نظام بحث هرمي شامل مستوحى من موقع netcarshow.com لتحسين تجربة البحث عن السيارات في السوق البلغاري.

## السكريبتات المتاحة للاستخراج

### 1. `extract-netcarshow-data.js` - الاستخراج الشامل الآلي
**الاستخدام:** استخراج جميع البيانات تلقائياً من الصفحة الرئيسية
- يستخرج الشركات والموديلات والأجيال وأنواع الهيكل
- يعمل تلقائياً مع فترات انتظار لتجنب الحظر
- يحفظ البيانات في ملف JSON

**كيفية الاستخدام:**
1. اذهب إلى https://www.netcarshow.com/
2. افتح Developer Tools (F12) → Console
3. انسخ محتوى `extract-netcarshow-data.js` والصقه
4. اضغط Enter
5. انتظر حتى ينتهي الاستخراج (قد يستغرق وقتاً طويلاً)

### 2. `collect-all-links.js` - جمع الروابط
**الاستخدام:** جمع جميع روابط الشركات والموديلات
- يجمع جميع الروابط المتاحة في الصفحة
- مفيد لمعرفة نطاق البيانات المتاحة
- يساعد في التخطيط للاستخراج

**كيفية الاستخدام:**
1. اذهب إلى https://www.netcarshow.com/
2. افتح Console والصق الكود
3. انسخ الروابط من Console للاستخدام اللاحق

### 3. `extract-single-make.js` - استخراج شركة واحدة
**الاستخدام:** استخراج بيانات شركة محددة بالتفصيل
- مثال: https://www.netcarshow.com/bmw/
- يستخرج جميع موديلات الشركة
- أسرع من الاستخراج الشامل

### 4. `extract-model-details.js` - استخراج موديل محدد
**الاستخدام:** استخراج تفاصيل موديل مع الأجيال وأنواع الهيكل
- مثال: https://www.netcarshow.com/bmw/i8/
- يستخرج الأجيال والسنوات وأنواع الهيكل

## خطوات الاستخراج الموصى بها

### الطريقة السريعة:
1. شغل `collect-all-links.js` لمعرفة الروابط المتاحة
2. استخدم `extract-single-make.js` لكل شركة رئيسية
3. استخدم `extract-model-details.js` للموديلات المهمة

### الطريقة الشاملة:
1. شغل `extract-netcarshow-data.js` للاستخراج التلقائي الكامل
2. (قد يستغرق ساعات حسب عدد الشركات)

## نصائح مهمة

### تجنب الحظر:
- استخدم فترات انتظار بين الطلبات
- لا تشغل عدة سكريبتات في نفس الوقت
- استخدم VPN إذا لزم الأمر

### تحسين الأداء:
- ابدأ بالشركات الرئيسية (BMW, Mercedes, Audi, VW)
- استخرج الموديلات الأكثر شيوعاً أولاً
- استخدم البيانات المؤقتة للاختبار

### استكشاف الأخطاء:
- تأكد من أنك في الصفحة الصحيحة
- تحقق من Console للأخطاء
- جرب السكريبتات على شركات مختلفة

## المتطلبات
- متصفح حديث مع Developer Tools
- اتصال إنترنت مستقر
- صبر للاستخراج الشامل (ساعات)

## الدعم
إذا واجهت مشاكل، تأكد من:
1. نسخ الكود كاملاً إلى Console
2. التحقق من أن الموقع لم يتغير هيكله
3. تجربة السكريبتات على صفحات مختلفة
