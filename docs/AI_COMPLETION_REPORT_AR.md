### 2. خدمة تحسين التكلفة (Cost Optimizer) ✅
**الملف:** `src/services/ai/ai-cost-optimizer.service.ts` (121 سطر)

**الوظيفة:**

- حدود الميزانية: 50$ لـ Gemini، 30$ لـ DeepSeek
- يعيد ضبط العدادات تلقائياً في بداية كل شهر

**قاعدة البيانات:**
```
ai_cost_tracking/monthly_stats:
  gemini: {
    monthlySpent: رقم (الإنفاق الشهري),
    requestCount: رقم (عدد الطلبات),
    lastReset: "2025-01" (آخر إعادة ضبط),
    lastUpdate: وقت التحديث
  }
  deepseek: { نفس البنية }
```

### 3. خدمة DeepSeek المحسنة (6 ميزات متقدمة) ✅
**الملف:** `src/services/ai/deepseek-enhanced.service.ts`

#### الميزة 1: اقتراح السعر الذكي 💰
```typescript
const result = await deepSeekEnhancedService.suggestPrice(carData, 'bg');
// يُرجع: { suggestedPrice, priceRange: {min, max}, reasoning, confidence }
```
- يحلل سوق السيارات البلغاري
- يعطي نطاق سعر (أدنى/أعلى)
- يشرح السبب بالبلغارية أو الإنجليزية

#### الميزة 2: تحليل الصور 📸
```typescript
const result = await deepSeekEnhancedService.analyzeCarImages(imageUrls, 'bg');
```
- حالياً: عنصر نائب (DeepSeek Vision API غير متوفر بعد)
- مستقبلاً: سيحلل حالة السيارة، الأضرار، المميزات

#### الميزة 3: مساعد البحث 🔍
```typescript
const result = await deepSeekEnhancedService.searchAssistant('أبحث عن BMW بأقل من 20 ألف', 'bg');
// يُرجع: { interpretation, suggestedFilters, additionalQuestions }
```
- يفهم اللغة الطبيعية (البلغارية/العربية/الإنجليزية)
- يحول الاستفسار إلى فلاتر منظمة
- يقترح أسئلة توضيحية

#### الميزة 4: فاحص جودة الإعلان ✅
```typescript
const result = await deepSeekEnhancedService.checkListingQuality(listing, 'bg');
// يُرجع: { score: 0-100, strengths, weaknesses, suggestions }
```
- يحسب درجة الجودة من 0-100
- يحدد نقاط القوة والضعف
- يقترح تحسينات محددة

#### الميزة 5: كشف الاحتيال 🚨
```typescript
const result = await deepSeekEnhancedService.detectFraud(listing, 'bg');
// يُرجع: { riskLevel: 'low'|'medium'|'high', confidence, warnings, reasoning }
```
- يكتشف الأعلام الحمراء (السعر غير الواقعي، معلومات متناقضة)
- يعطي مستوى المخاطرة (منخفض/متوسط/عالي)
- يوضح السبب بالتفصيل

#### الميزة 6: مساعد الدردشة 💬
```typescript
const result = await deepSeekEnhancedService.chatbot(message, 'listing', 'bg');
// السياقات: 'general' | 'listing' | 'search' | 'support'
```
- يفهم السياق (عام/إعلان/بحث/دعم)
- يجيب باللغة البلغارية أو الإنجليزية
- يقدم مساعدة احترافية

});
