## 📦 **التفاصيل التقنية:**

```css
padding: 0.625rem 1rem;  /* صغير */
display: inline-block;   /* حجم مضغوط */
margin: 0.5rem 0 1rem 0; /* مسافة صغيرة */
```

### **Price (محدّث):**
```css
font-size: 1.5rem;  /* 24px بدلاً من 48px */
display: inline;    /* inline مع PriceLabel */
```

### **PriceLabel (محدّث):**
```css
font-size: 0.75rem;  /* 12px صغير */
display: span;       /* inline */
margin-left: 0.5rem; /* بجانب السعر */
```

```css
جميع العناوين والنصوص:
```
