## 🚀 كيفية الاستخدام

### الخطوة 1: بناء المشروع
```bash
cd bulgarian-car-marketplace
npm run build
```

### الخطوة 2: النشر على Firebase
```bash
firebase deploy
```

### الخطوة 3: الاختبار
1. افتح الموقع على Chrome Mobile
2. اضغط على القائمة (⋮)
3. اختر "Install App" أو "Add to Home Screen"
4. التطبيق سيظهر على الشاشة الرئيسية
