## 🐛 الأخطاء البرمجية

### 1. مشكلة CSS Specificity

**في:** `mobile-responsive.css`

```css
/* ❌ خطأ: استخدام !important بكثرة */
button {
  min-height: 44px !important;
  padding: 12px 18px !important;
  font-size: 1rem !important;
}

/* ❌ خطأ: تعارض مع styled-components */
.card {
  padding: 20px !important;
}
```

**المشكلة:**
- `!important` يتعارض مع styled-components
- صعوبة override الأنماط
- تضارب في الأولويات

---

### 2. مشكلة Z-Index

**في:** `Header.tsx` و `MobileHeader.tsx`

```typescript
// Header.tsx
z-index: 1000

// MobileHeader.tsx  
z-index: 1000

// mobile-design-system.ts
mobileZIndex = {
  sticky: 1020,
  modal: 1050
}
```

**المشكلة:**
```
❌ قيم z-index غير متسقة
❌ احتمال تداخل العناصر
❌ عدم استخدام نظام z-index الموحد
```

---

### 3. مشكلة Body Padding

**في:** `mobile-responsive.css`

```css
@media (max-width: 768px) {
  body {
    padding-top: 80px !important;
  }
  
  body > #root {
    padding-top: 0; /* تعليق: تم إزالته! */
  }
}
```

**المشكلة:**
```
❌ padding-top مضاف مرتين
❌ تعليقات متضاربة
❌ قد يسبب مسافة زائدة
```

---

### 4. مشكلة Font Size في iOS

**في:** `mobile-design-system.ts`

```typescript
preventZoom: css`
  font-size: 16px;
  @supports (-webkit-touch-callout: none) {
    font-size: max(16px, 1rem); // ❌ تكرار غير ضروري
  }
`
```

**المشكلة:**
- `16px` و `1rem` نفس القيمة (إذا كان root = 16px)
- `max()` غير ضروري
