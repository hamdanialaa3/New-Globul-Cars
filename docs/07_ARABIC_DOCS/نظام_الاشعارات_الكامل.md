```bash
cd bulgarian-car-marketplace
npm install firebase
```

### الخطوة 2: تفعيل Firebase Messaging
```bash
# في Firebase Console:
1. اذهب لـ Project Settings
2. Cloud Messaging
3. انسخ "Server Key"
4. انسخ "Sender ID"
```

### الخطوة 3: إضافة البانر في App.tsx
```typescript
import { NotificationBanner } from './components/Notifications/NotificationBanner';

function App() {
  return (
    <>
      <NotificationBanner />
      {/* باقي التطبيق */}
    </>
  );
}
```

### الخطوة 4: تهيئة الإشعارات
```typescript
// في App.tsx أو index.tsx
import { notificationService } from './services/notification-service';

useEffect(() => {
  notificationService.initialize();
}, []);
```

### الخطوة 5: نشر Cloud Functions
```bash
cd functions
npm install
firebase deploy --only functions
```
