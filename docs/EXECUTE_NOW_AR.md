## 🐛 إذا حدث خطأ

### ❌ المشكلة: فشل سكريبت المزامنة
**الحل:**
```bash
# تحقق من بيانات اعتماد Firebase
echo %FIREBASE_SERVICE_ACCOUNT_KEY%

# إذا كان فارغاً، تحقق من .env.local:
FIREBASE_SERVICE_ACCOUNT_KEY={"type":"service_account",...}
```

### ❌ المشكلة: "Algolia not initialized"
**الحل:**
```bash
# تحقق من أن .env.local يحتوي على هذه الأسطر:
REACT_APP_ALGOLIA_APP_ID=RTGDK12KTJ
REACT_APP_ALGOLIA_SEARCH_API_KEY=01d60b828b7263114c11762ff5b7df9b
REACT_APP_ALGOLIA_INDEX_NAME=cars

# ثم أعد تشغيل السيرفر
npm start
```

### ❌ المشكلة: "Index does not exist"
**الحل:**
```bash
# نفّذ المزامنة مرة أخرى - ستنشئ الفهرس
node scripts/sync-algolia.js
```

### ❌ المشكلة: البحث لا يزال بطيئاً (800ms)
**تحقق من:**
1. Browser Network tab → ترى استدعاءات algolia.net؟
2. الكونسول → ترى رسالة "Algolia initialized"؟
3. لوحة Algolia → ترى سجلات في فهرس "cars"؟
