## 🔄 How to Update Documentation
1. **New Feature:** Add a guide in `docs/features/` or `docs/guides/`.
2. **Architecture Change:** Update `docs/architecture/` and `CONSTITUTION.md`.
3. **Audit/Fix:** Save reports to `docs/reports/`.
4. **AI Logs:** Move chat logs to `docs/ai_history/`.
