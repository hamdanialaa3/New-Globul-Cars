## سيناريوهات المستخدمين

### 👨 سيناريو 1: مريم - شاب يبحث عن سيارته الأولى

**الملف الشخصي:**
- العمر: 28 سنة
- الموقع: صوفيا، بلغاريا
- الميزانية: 8,000 - 12,000 EUR
- الاهتمام: سيارة عائلية اقتصادية

**الرحلة:**

#### الخطوة 1: البحث الأولي
```
1. يفتح الموقع من Google
2. يشاهد الصفحة الرئيسية
3. يختار "البحث عن سيارة" من الشريط العلوي
```

**ما يحتاج المشروع أن يوفره:**
- شريط بحث سريع وسهل
- صفحة رئيسية احترافية
- روابط واضحة وسهلة الفهم

---

#### الخطوة 2: استخدام البحث المتقدم
```
يدخل /advanced-search
- يختار نوع السيارة: "سيارات ركاب"
- يختار ماركة: "Toyota"
- ينظر نطاق السعر: 8,000 - 12,000 EUR
- يختار الموقع: "Sofia"
```

**الفلاتر المهمة:**
- نوع السيارة
- نطاق السعر
- السنة (2015 أو أحدث)
- حالة الوقود (لا يهم)

---

#### الخطوة 3: عرض النتائج
```
النتائج: 45 سيارة توافق المعايير

يرى:
- صورة السيارة الرئيسية
- الماركة والموديل والسنة
- السعر
- المسافة المقطوعة
- الموقع
- تقييم البائع
```

**التصميم المطلوب:**
- بطاقة واضحة
- صور عالية الجودة
- معلومات مختصرة وأساسية فقط

---

#### الخطوة 4: مشاهدة التفاصيل
```
ينقر على سيارة معينة ("/car/1/5")

يشاهد:
- معرض الصور (10+ صور)
- البيانات الكاملة
- الميزات والتجهيزات
- بيانات البائع
- التقييمات
```

**التفاعلات:**
- تكبير الصور
- عرض الخريطة
- قراءة التقييمات
- الاتصال بالبائع

---

#### الخطوة 5: الاتصال بالبائع
```
ينقر على "إرسال رسالة"

خيارات:
1. تسجيل دخول (إذا لم يكن مسجلاً)
2. كتابة الرسالة
3. إرسال الرسالة
```

**التسلسل:**
```
يدخل /login
    ↓
يكتب البريد والكلمة
    ↓
يتم توجيهه إلى /messages
    ↓
الرسالة تُرسل للبائع
    ↓
البائع يتلقى إشعار
```

### 👤 سيناريو 2: أحمد - تاجر يبيع سياراته

**الملف الشخصي:**
- الوظيفة: تاجر سيارات
- عدد السيارات: 10-20
- الخبرة: 5 سنوات
- الهدف: بيع سريع وإدارة سهلة

**الرحلة:**

#### الخطوة 1: تسجيل حساب تاجر
```
يدخل /register

يختار: "حساب تاجر"

يملأ:
- الاسم الكامل
- البريد الإلكتروني
- كلمة المرور
- رقم الهاتف
- المدينة
- اسم المحل (المعرض)
```

**الفرق من حساب عادي:**
- حقول إضافية للمحل
- خيارات فريق (Team)
- إحصائيات متقدمة

---

#### الخطوة 2: إنشاء إعلان أول
```
يدخل /sell

يختار: "ابدأ الآن"

ينتقل إلى: /sell/auto
    ↓
يختار نوع السيارة: "passenger_cars"
    ↓
يختار النوع: "تاجر"
    ↓
يملأ البيانات الأساسية
    ↓
يضيف التجهيزات
    ↓
يرفع الصور (15 صورة)
    ↓
يعيّن السعر
    ↓
يملأ بيانات الاتصال
    ↓
معاينة الإعلان
    ↓
نشر الإعلان
```

**المتوقع:**
- عملية سريعة (10-15 دقيقة)
- حفظ تلقائي للمسودات
- اقتراحات السعر من AI

---

#### الخطوة 3: إدارة الإعلانات
```
يدخل /profile/my-ads

يرى:
- جميع إعلاناته (نشط، معلق، انتهى)
- عدد المشاهدات لكل سيارة
- عدد الرسائل
- عدد الإعجابات
```

**الأزرار المتاحة:**
- تعديل الإعلان
- تجديد الإعلان (دفع)
- حذف الإعلان
- إعادة نشر الإعلان
- عرض الإحصائيات

---

#### الخطوة 4: الرسائل والتفاوض
```
يدخل /messages

يرى:
- قائمة المحادثات
- آخر رسالة في كل محادثة
- عدد الرسائل الجديدة (غير المقروءة)
```

**المحادثة:**
```
يفتح محادثة مع المشتري الجديد

يرى:
- التاريخ الكامل للرسائل
- صورة المشتري
- بيانات الاتصال
```

**التفاوض:**
```
المشتري: "هل تقبل 10,500 بدلاً من 11,000؟"

التاجر: "السعر 10,800 الحد الأدنى"

المشتري: "تمام، متى يمكنني الرؤية؟"

التاجر: "الجمعة الساعة 5 العصر"
```

---

#### الخطوة 5: الإحصائيات والتحليل
```
يدخل /profile/analytics

يشاهد:
- عدد المشاهدات اليومي
- معدل التحويل
- أكثر الساعات ازدحاماً
- أكثر الكلمات بحثاً
```

**الفائدة:**
- فهم السوق
- تحسين الأسعار
- تحسين الأوصاف
- تحديد أفضل أوقات النشر

### 🏢 سيناريو 3: محمد - مسؤول الموقع

**الملف الشخصي:**
- الوظيفة: مسؤول الموقع
- الصلاحيات: Admin
- المسؤوليات: إدارة المستخدمين والمحتوى

**الرحلة:**

#### الخطوة 1: الدخول إلى لوحة الإدارة
```
يدخل /admin-login

يكتب:
- البريد الإلكتروني
- كلمة المرور

يدخل /admin
```

**ما يراه:**
- لوحة معلومات رئيسية
- إحصائيات عامة
- أحدث الأنشطة

---

#### الخطوة 2: التحقق من الإعلانات الجديدة
```
يدخل قسم إدارة السيارات

يرى:
- قائمة الإعلانات الجديدة (pending)
- بيانات المالك
- تفاصيل السيارة
- الصور
```

**الإجراء:**
```
يفتح إعلان

يتحقق من:
- صحة الصور
- صحة البيانات
- تطابق السعر
- عدم وجود مشاكل

يختار:
✅ "Approve" - يُنشر الإعلان
❌ "Reject" - مع سبب الرفض
⏳ "Pending" - للتحقق لاحقاً
```

---

#### الخطوة 3: إدارة المستخدمين المشبوهين
```
يدخل /admin/users

يبحث عن مستخدم

يرى:
- عدد الإعلانات
- عدد الشكاوى
- سلوك المستخدم
- التقييمات
```

**الإجراء:**
```
إذا كان هناك شكاوى متكررة:

1. يفتح ملف المستخدم
2. يراجع الشكاوى
3. يتواصل مع المستخدم (رسالة تحذير)
4. إذا استمر: يحظر الحساب
```

---

#### الخطوة 4: تحليل الأداء
```
يدخل /admin/analytics

يشاهد:
- عدد المستخدمين الجدد
- عدد الإعلانات الجديدة
- الإيرادات اليومية
- معدل الرضا
```

**القرارات:**
```
يرى أن:
- المستخدمين يزيدون 5% أسبوعياً ✅
- الإعلانات تقل 2% أسبوعياً ⚠️
- الإيرادات مستقرة

يقرر:
- زيادة الحملات الترويجية
- تحسين تجربة البائع
```

## السيناريوهات الشائعة

### السيناريو 1: ماذا لو نسي المستخدم كلمة المرور؟

**التسلسل:**
```
1. يفتح /login
2. ينقر "نسيت كلمة المرور؟"
3. يدخل بريده الإلكتروني
4. يتلقى رسالة بريد بها رابط تصفير
5. ينقر الرابط
6. يدخل كلمة مرور جديدة
7. يتم تسجيل الدخول تلقائياً
```

**الرسالة المتوقعة:**
```
الموضوع: إعادة تعيين كلمة المرور - Koli One

مرحباً [الاسم]،

تلقينا طلبك لإعادة تعيين كلمة المرور.

انقر الرابط أدناه لتعيين كلمة مرور جديدة:
[رابط آمن صالح 24 ساعة]

إذا لم تطلب هذا، تجاهل هذه الرسالة.

فريق Koli One
```

---

### السيناريو 2: ماذا لو كانت الصور بجودة سيئة؟

**عند الرفع:**
```
1. المستخدم يرفع صورة بجودة منخفضة
2. النظام يكتشفها (أقل من 800×600)
3. يعرض تحذير: "جودة الصورة منخفضة، الرجاء رفع صورة أفضل"
4. يقترح: "استخدم كاميرا أفضل أو صورة موجودة"
```

**عند النشر:**
```
1. إذا كانت جميع الصور سيئة: رفض الإعلان
2. إذا بعضها جيد: موافقة مع ملاحظة
3. التحويل التلقائي إلى WebP (ضغط)
4. إعادة حجم تلقائية للعرض الأمثل
```

---

### السيناريو 3: مستخدم يريد حذف حسابه

**التسلسل:**
```
1. يدخل /profile/settings
2. ينقر "حذف الحساب"
3. يظهر تحذير: "هذا الإجراء غير قابل للتراجع"
4. ينقل إلى /data-deletion
5. يملأ نموذج مع السبب
6. ينقر "تأكيد الحذف"
7. يتلقى رسالة بريد للتأكيد
8. ينقر الرابط في البريد
9. يتم حذف جميع بيانات المستخدم
```

**البيانات المحذوفة:**
- ملف الفرد
- الإعلانات
- الرسائل
- المفضلات
- الإشعارات
- سجل الأنشطة (محفوظ للأرشيفة)

---

### السيناريو 4: تاجر يريد ترقية خطته

**التسلسل:**
```
1. يدخل /profile
2. يرى الخطة الحالية: "Dealer - 5 إعلانات نشطة من 30"
3. يرى رسالة: "هل تريد المزيد من الإعلانات؟"
4. ينقر "الترقية إلى Company"
5. ينقل إلى /billing
6. يرى السعر: 100€/شهر
7. ينقر "الدفع الآن"
8. ينقل إلى /checkout
9. يختار طريقة الدفع (Stripe)
10. يكمل عملية الدفع
11. يتلقى تأكيد البريد
12. الخطة محدثة فوراً
13. يمكنه الآن نشر إعلانات غير محدود
```

---

### السيناريو 5: مستخدم يريد الإبلاغ عن إعلان مزيف

**التسلسل:**
```
1. يفتح صفحة السيارة (/car/1/5)
2. ينقر "الإبلاغ عن هذا الإعلان" (ثلاث نقاط)
3. يختار السبب:
   - صورة مزيفة
   - سعر غير صحيح
   - سيارة غير موجودة
   - احتيال
   - أخرى
4. يكتب التفاصيل
5. يرفع صورة أو دليل (إن أمكن)
6. ينقر "إرسال"
7. يتم إرسال التقرير إلى Admin
8. Admin يتحقق (خلال 24 ساعة)
9. إذا تم التأكيد: يتم حذف الإعلان وحظر البائع
```

## معالجة الأخطاء والاستثناءات

### 🔴 الأخطاء الشائعة

#### 1. خطأ: "البريد الإلكتروني مستخدم بالفعل"
```
السبب: البريد موجود بالفعل

الحل:
1. اذهب إلى /login
2. انقر "نسيت كلمة المرور"
3. أعد تعيين كلمة المرور
4. سجل دخول
```

#### 2. خطأ: "البيانات غير كاملة"
```
السبب: حقل إجباري مفقود

الحل:
- ملأ جميع الحقول الإجبارية (بعلامة *)
- تحقق من صيغة البريد الإلكتروني
- تحقق من صيغة رقم الهاتف
```

#### 3. خطأ: "الصور كبيرة جداً"
```
السبب: حجم الملف يتجاوز 5MB

الحل:
- ضغط الصور قبل الرفع
- استخدام موقع مثل TinyPNG
- النظام يفعل هذا تلقائياً الآن
```

#### 4. خطأ: "انتهت جلسة الدخول"
```
السبب: نسيت الموقع مفتوحاً يومين

الحل:
- سجل الدخول مجدداً
- البيانات محفوظة (لا تفقد أي شيء)
```

## نصائح وحيل للاستخدام

### 💡 للمشترين

1. **استخدم فلاتر متعددة**
   - ضيق النطاق للعثور على السيارة المثالية
   - ادخر الوقت بدلاً من تصفح 500 سيارة

2. **احفظ المفضلة**
   - أضف السيارات التي تعجبك للمفضلة
   - قارن بينهم لاحقاً

3. **اقرأ التقييمات**
   - تقييمات البائع مهمة جداً
   - تجنب البائعين ذوي التقييمات السلبية

4. **تفاوض بشكل معقول**
   - لا تسأل عن خصم 50%
   - 5-10% خصم معقول

5. **استخدم الخريطة**
   - تحقق من موقع السيارة
   - احسب المسافة

### 💡 للبائعين

1. **أضف صور عالية الجودة**
   - 10-15 صورة من زوايا مختلفة
   - الصور الجيدة = بيع أسرع

2. **اكتب وصف دقيق**
   - اذكر أي عيوب (الشفافية أولاً)
   - استخدم الوصف الذي توليده AI

3. **سعر تنافسي**
   - استخدم أداة اقتراح السعر
   - قارن مع إعلانات مشابهة

4. **استجب للرسائل سريعاً**
   - الرد في ساعة = بيع أسرع
   - المشترين ينسون بسهولة

5. **أدرج جميع التفاصيل**
   - التاريخ والمسافة والحالة
   - التاريخ الدقيق للتفتيش الأخير

### 💡 للتجار

1. **أدر فريم فعال**
   - أضف موظفين لمعالجة الرسائل
   - صفحة شركتك هي أهم شيء

2. **استخدم الحملات الإعلانية**
   - روج الإعلانات الشهيرة
   - استثمر في الظهور

3. **راقب الإحصائيات**
   - اسأل نفسك: لماذا لم تُبع هذه السيارة؟
   - حسّن الأسعار والصور والأوصاف

4. **بناء سمعة**
   - الرد بسرعة والاحترافية
   - تقييمات جيدة = عملاء أكثر
