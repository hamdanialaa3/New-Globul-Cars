# حالة تنفيذ الذكاء الاصطناعي (صفر تكلفة)

آخر تحديث: 20 نوفمبر 2025

## 1. نظرة عامة مختصرة
تم بناء طبقة ذكاء اصطناعي بدون تكلفة شهرية اعتمادًا على Firebase (Spark) + نموذج Gemini المجاني (gemini-1.5-flash). تم تنفيذ وظائف خلفية (Cloud Functions) لمعالجة البيانات، تحسين الصور، كشف التكرار، وإرسال التنبيهات، مع طبقة أحداث Analytics.

## 4. الأسرار المطلوبة (لا تُخزن في Git)
استخدم أوامر Firebase Functions Config بدلًا من حفظ القيم في المستودع.

```powershell
firebase functions:config:set \
  email.smtp_user="globul.net.m@gmail.com" \
  email.smtp_pass="<APP_PASSWORD_جديد_بعد_التدوين>" \
  alerts.admin_email="globul.net.m@gmail.com" \
  gemini.api_key="<GEMINI_API_KEY>" \
  gemini.model="gemini-1.5-flash" \
  ai.display_name_bg="Clover Alina" \
  ai.display_name_en="Clover Alina"
```

ثم:
```powershell
firebase deploy --only functions
```

مهم: كلمة مرور التطبيق المكشوفة في المحادثة يجب تدويرها الآن وإنشاء App Password جديد ثم استخدامه أعلاه.

## 5. ملاحظات حول نموذج Gemini
- النموذج المستخدم: `gemini-1.5-flash` (سريع وملائم لتكلفة صفرية).
- يمكن لاحقًا تبديل `gemini.model` في Config لنموذج أعلى دقة إذا لزم.

## 6. دمج قواعد Firestore
القواعد الحالية (حالياً في `firestore.rules`) معقدة وتشمل: أدوار إدارية، حدود خطط، شركات، معارض سيارات، محادثات، منشورات، تكامل Stripe.

القواعد المبسطة (المسودة في `firestore-enhanced.rules`) تُقلّص السطح الأمني بشكل كبير لكنها تزيل تحققات حيوية:
- لا تحققات خطة أو حدود إدراج.
- لا تمييز أدوار إدارية.
- لا حماية خاصة لمحادثات أو سجلات دفع.

### خيارات الدمج المقترحة:
1. إبقاء القواعد الحالية كما هي (آمن) – ودمج فقط إضافة مسار الصور إذا لزم.
2. إنشاء نسخة تجريبية: نسخ المبسطة إلى `firestore.sandbox.rules` واختبارها عبر المحاكي.
3. دمج تدريجي: الحفاظ على جميع أقسام الأمان الحساسة، وإضافة تحسينات انتقائية (مثل تبسيط قسم `cars` فقط).

لا يُنصح باستبدال مباشر شامل إلا مع خطة أمان بديلة واضحة. الرجاء التأكيد الكتابي قبل أي استبدال.

## 7. تتبع الأداء (Frontend)
استخدام الملف: `ai-performance-traces.ts`:
```typescript
import { withAiTrace, AI_TRACE_NAMES } from '@/services/performance/ai-performance-traces';
await withAiTrace(AI_TRACE_NAMES.geminiInference, () => runGeminiPrompt(payload));
```
أضف الاستدعاءات في نقاط: (1) استعلام تسعير، (2) تحسين صورة، (3) استدلال فئة.
