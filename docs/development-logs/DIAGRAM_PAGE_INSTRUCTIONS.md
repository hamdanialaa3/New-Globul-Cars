## 🎯 المميزات

### 1. المربعات (Nodes)
- كل package يمثل بمربع ملون
- 11 مربع بألوان مميزة
- يمكن سحبها وتحريرها

### 2. الخطوط (Links)
- تمثل الوصلات البرمجية
- أسهم توضح اتجاه التبعية
- تفاعلية عند التمرير

### 3. التفاعل
- **سحب**: اسحب أي package لتحريكه
- **النقر**: انقر لعرض التفاصيل
- **التصفية**: استخدم الأزرار
- **إعادة تعيين**: اضغط Reset
