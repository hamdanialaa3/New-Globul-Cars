# Bulgarian Car Marketplace - Quick Start Guide

## 🚀 التشغيل السريع (الطريقة الأسهل)

### الطريقة الأولى: استخدم الملف الجاهز
1. اضغط مرتين على `Quick-Start-Fixed.bat` في المجلد الرئيسي
2. انتظر حتى ينتهي البناء والتثبيت
3. المشروع سيفتح تلقائياً على `http://localhost:3001`

### الطريقة الثانية: التشغيل اليدوي
```bash
# انتقل للمجلد
cd bulgarian-car-marketplace

# ثبت التبعيات
npm install

# شغل الخادم المستقر
npm run start:stable
```

## 🔧 حل المشاكل الشائعة

### مشكلة ESLint Plugin
```bash
# إذا ظهرت مشاكل مع ESLint
npm uninstall eslint-webpack-plugin
npm run start:stable
```

### مشكلة الخادم لا يعمل
```bash
# استخدم الخادم البديل
npm run start:python
# أو
npm run serve:static
```

### مشكلة البناء
```bash
# استخدم البناء المباشر
npm run build
python -m http.server 3001 -d build
```

## 📋 الميزات المتاحة

- 🏠 **الصفحة الرئيسية**: تصميم احترافي أسود وأصفر
- 🔍 **نظام البحث المتقدم**: بحث متقدم عن السيارات
- 👤 **لوحة تحكم البائع**: إدارة السيارات والإحصائيات
- 💬 **نظام المراسلة**: تواصل بين البائعين والمشترين
- 🔐 **المصادقة**: تسجيل دخول آمن
- 🌍 **دعم البلغارية**: واجهة كاملة باللغة البلغارية

## ⚡ نصائح للاستقرار

1. **استخدم دائماً** `npm run start:stable` للتشغيل المستقر
2. **إذا لم يعمل**: استخدم `Quick-Start-Fixed.bat`
3. **للتطوير**: استخدم `npm run start:dev` (قد يحتوي على مشاكل)
4. **للإنتاج**: استخدم `npm run build` ثم `python -m http.server 3001 -d build`

## 🔍 استكشاف الأخطاء

### إذا لم يفتح المتصفح
- تأكد من أن المنفذ 3001 غير مستخدم
- جرب فتح `http://localhost:3001` يدوياً

### إذا ظهرت أخطاء في البناء
- احذف مجلد `node_modules` و `package-lock.json`
- أعد تثبيت: `npm install`
- شغل البناء مرة أخرى

### إذا توقف الخادم
- أعد تشغيل `Quick-Start-Fixed.bat`
- أو استخدم `npm run start:python`
