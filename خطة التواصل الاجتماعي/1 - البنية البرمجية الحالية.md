## 4. Firestore Collections

### الموجود حالياً:

**users/**
```
{userId}/
  - displayName
  - profileImage
  - profileType ('private' | 'dealer' | 'company')
  - trustScore
  - isVerified
  - stats { followers, following, posts }
```

**posts/**
```
{postId}/
  - authorId
  - authorInfo { ... }
  - type ('text' | 'car_showcase' | 'tip' | 'question' | 'review')
  - content { text, media, carReference, hashtags }
  - visibility ('public' | 'followers' | 'private')
  - engagement { views, likes, comments, shares, saves }
  - status ('active' | 'hidden' | 'deleted')
  - createdAt
  - updatedAt
```

**follows/**
```
{followId}/
  - followerId
  - followingId
  - createdAt
```

**reactions/**
```
{postId}/
  likes/
    {userId}: timestamp
```

**comments/**
```
{commentId}/
  - postId
  - authorId
  - text
  - createdAt
```

**cars/**
```
{carId}/
  - للربط مع البوستات
```

---

### مطلوب إضافة:

**feeds/** (جديد)
```
{userId}/
  personalFeed/
    {feedItemId}/
      - postId
      - score
      - addedAt
      - reason ('trending' | 'following' | 'similar')
```

**postScores/** (جديد)
```
{postId}/
  - totalScore
  - engagementScore
  - recencyScore
  - qualityScore
  - authorScore
  - updatedAt
```

**userInterests/** (جديد)
```
{userId}/
  - carBrands: Map<string, number>
  - locations: Map<string, number>
  - hashtags: Map<string, number>
  - authors: Map<string, number>
  - updatedAt
```

**postEngagements/** (تطوير)
```
{postId}/
  reactions/
    {userId}: reactionType (7 types)
  comments/ (threaded)
    {commentId}/
      - text
      - authorId
      - parentCommentId
      - createdAt
  shares/
    {userId}: sharedAt
  saves/
    {userId}: savedAt
```
