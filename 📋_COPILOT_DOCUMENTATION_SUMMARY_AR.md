## الإجابة

ملف `.github/copilot-instructions.md` هو **دليل شامل** لمساعد الذكاء الاصطناعي GitHub Copilot لفهم بنية مشروع New Globul Cars واتفاقياته البرمجية.

### ما هو GitHub Copilot؟
GitHub Copilot هو مساعد ذكاء اصطناعي من Microsoft/GitHub يساعد المطورين في كتابة الكود. يقرأ ملفات التكوين في المشروع ليقدم اقتراحات ذكية متوافقة مع معايير المشروع.

### لماذا نحتاج هذا الملف؟
- ✅ **للمطورين:** مرجع شامل لفهم بنية المشروع
- ✅ **لـ Copilot AI:** تعليمات لتقديم اقتراحات كود متوافقة مع المعايير
- ✅ **للفريق:** ضمان اتباع الجميع لنفس الاتفاقيات
- ✅ **للجودة:** الحفاظ على تناسق الكود
