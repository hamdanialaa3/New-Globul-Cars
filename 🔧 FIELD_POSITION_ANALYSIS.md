```
حقل "ГЕОРГИЕВА" (الاسم الأوسط البلغاري) يغطي العين اليسرى!
```

## 📊 التحليل:

### الصورة الأصلية:
- **الملف:** `ID_front (1).png`
- **الحجم الأصلي:** يجب قياسه
- **التخطيط:** 
  - الصورة الشخصية على اليمين
  - البيانات النصية على اليسار

### المشكلة:
حقل `ГЕОРГИЕВА` موضوع عند:
```typescript
{
  id: 'middleNameBG',
  position: { x: 407, y: 189, width: 130, height: 30 }
}
```

هذا بعيد جداً لليمين ويتداخل مع الصورة الشخصية!

### الحل:
يجب تصحيح المواقع بناءً على التخطيط الفعلي للبطاقة البلغارية.
