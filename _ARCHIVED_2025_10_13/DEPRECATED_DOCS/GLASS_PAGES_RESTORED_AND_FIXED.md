## 🧪 **الآن جرب:**

### **الخطوة 1: امسح كل شيء**
```
في Chrome:
1. Ctrl + Shift + Delete
2. اختر "All time"
3. ✓ Cookies and other site data
4. ✓ Cached images and files
5. اضغط "Clear data"
```

### **الخطوة 2: أعد تشغيل الخادم**
```
في Terminal:
Ctrl + C (إيقاف الخادم)
npm start (بدء من جديد)
```

### **الخطوة 3: افتح في Incognito**
```
Ctrl + Shift + N
ثم اذهب إلى:
http://localhost:3000/login
```

### **الخطوة 4: جرب Google Sign-In**
```
اضغط F12 → Console
اضغط على زر Google
شاهد ما يحدث في Console
```
