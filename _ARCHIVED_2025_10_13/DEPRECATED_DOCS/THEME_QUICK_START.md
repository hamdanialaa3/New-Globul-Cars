# 🎨 دليل التحكم في مظهر Globul Cars

## الآن لديك السيطرة الكاملة على مظهر مشروعك! 🚀

### ما تم إنجازه:
✅ **نظام مظهر مركزي** - جميع الألوان في مكان واحد
✅ **دليل شامل** - `THEME_CONTROL_GUIDE.md` مع أمثلة مفصلة
✅ **معاينة فورية** - `ThemePreview.tsx` لرؤية التغييرات
✅ **ألوان جاهزة** - `color-presets.json` مع مجموعات متنوعة
✅ **أداة تغيير سريع** - `change-theme.bat` لتطبيق الألوان بضغطة زر

---

## 🔧 كيفية الاستخدام السريع

### 1. تغيير المظهر بضغطة زر:
```bash
# شغل هذا الملف
change-theme.bat
```
اختر الرقم المطلوب وسيتم تطبيق المظهر تلقائياً!

### 2. التخصيص اليدوي:
1. افتح `bulgarian-car-marketplace/src/styles/theme.ts`
2. غير الألوان في قسم `colors`
3. احفظ الملف
4. شاهد التغييرات فوراً

### 3. معاينة التغييرات:
```bash
cd bulgarian-car-marketplace
npm start
```
افتح `http://localhost:3000` لرؤية المظهر الجديد

## 🎯 الألوان المتاحة للتغيير

| اللون | الاستخدام | المثال |
|-------|-----------|---------|
| `primary` | الأزرار الرئيسية، الروابط | `#FFD700` (أصفر) |
| `secondary` | العناصر الثانوية | `#FFA500` (برتقالي) |
| `accent` | التأكيدات الخاصة | `#FF6B35` (أحمر برتقالي) |
| `background` | خلفية الصفحات | `#FFFFFF` (أبيض) |
| `text` | ألوان النصوص | `#333333` (رمادي داكن) |
