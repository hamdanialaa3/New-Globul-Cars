# 🔗 تكوين n8n مع حسابك الحقيقي

## 📝 **المعلومات المطلوبة**

لتكوين المشروع مع حسابك الحقيقي، احتاج المعلومات التالية:

### 1. 🌐 **معلومات n8n Cloud**
- **URL حسابك**: (مثل: https://yourname.app.n8n.cloud)
- **API Key**: من إعدادات حسابك
- **اسم المستخدم**: عادة الإيميل
- **كلمة المرور**: (للدخول المباشر)

### 2. 🔐 **API Access**
للحصول على API Key:
```
1. اذهب لحسابك في n8n
2. Settings → API Keys
3. Create New API Key
4. انسخ الـ API Key
```

### 3. 🎯 **Webhook URLs**
سنحتاج لتحديث URLs الـ webhooks من:
```
http://localhost:5678/webhook/
```
إلى:
```
https://yourname.app.n8n.cloud/webhook/
```

## 🛠️ **خطوات التكوين**

### الخطوة 1: شارك معلومات حسابك
أرسل لي:
- رابط حسابك في n8n
- API Key (إن أمكن)

### الخطوة 2: سأقوم بتحديث الملفات التالية
- ✅ `src/services/n8n-integration.ts`
- ✅ `.env` files
- ✅ `n8n-workflows/*.json`
- ✅ Test pages URLs

### الخطوة 3: رفع Workflows لحسابك
- تصدير الـ workflows JSON
- استيرادها في حسابك
- تفعيل الـ webhooks
