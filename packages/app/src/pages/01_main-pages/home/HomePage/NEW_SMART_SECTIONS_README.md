# الأقسام الذكية الجديدة للصفحة الرئيسية
# New Smart Sections for HomePage

## نظرة عامة | Overview

تم إضافة ثلاثة أقسام ذكية جديدة إلى الصفحة الرئيسية مع خوارزميات متقدمة لتحليل البيانات وتقديم تجربة مستخدم محسّنة.

Three new smart sections have been added to the HomePage with advanced algorithms for data analysis and enhanced user experience.

---

## 1. تصنيفات المركبات | Vehicle Classifications Section

### الملف | File
`VehicleClassificationsSection.tsx`

### الوصف | Description
قسم يعرض جميع أنواع المركبات المتاحة مع إحصائيات حية وتحليل الاتجاهات.

Displays all available vehicle types with live statistics and trend analysis.

### الميزات الذكية | Smart Features

#### 1. حساب الإحصائيات في الوقت الفعلي
- **عدد المركبات المتاحة**: يتم حساب عدد المركبات المتاحة لكل نوع من قاعدة البيانات
- **متوسط السعر**: حساب متوسط أسعار المركبات بالعملة الأوروبية (EUR)
- **تحليل الاتجاهات**: مقارنة النشاط بين آخر 30 يوم والـ 30 يوم السابقة

#### 2. مؤشرات الاتجاه الذكية
- **صاعد (📈)**: زيادة بنسبة أكثر من 5%
- **هابط (📉)**: انخفاض بنسبة أكثر من 5%
- **مستقر (➡️)**: تغيير أقل من 5%

#### 3. الترتيب التلقائي
يتم ترتيب التصنيفات تلقائياً حسب الشعبية (عدد المركبات المتاحة)

### أنواع المركبات المدعومة
1. **سيارات ركاب** (car) - للاستخدام الشخصي
2. **سيارات دفع رباعي** (suv) - قوة وأداء عالي
3. **شاحنات صغيرة** (van) - للنقل والتجارة
4. **دراجات نارية** (motorcycle) - سرعة ومرونة
5. **شاحنات كبيرة** (truck) - للنقل الثقيل
6. **حافلات** (bus) - نقل الركاب

### الخوارزمية المستخدمة
```typescript
calculateVehicleStats(vehicleType: string) {
  // 1. جلب جميع المركبات من النوع المحدد
  // 2. حساب متوسط السعر

  //    - عدد المركبات المضافة في آخر 30 يوم
  //    - مقارنة مع الـ 30 يوم السابقة
  //    - حساب نسبة التغيير
}
```

## 2. الفئات الأكثر طلباً | Most Demanded Categories Section

### الملف | File
`MostDemandedCategoriesSection.tsx`

### الوصف | Description
قسم مدعوم بالذكاء الاصطناعي يعرض الفئات الأكثر طلباً بناءً على تحليل متعدد العوامل.

AI-powered section displaying most demanded categories based on multi-factor analysis.

### الميزات الذكية | Smart Features

#### 1. خوارزمية الطلب الذكية
يتم حساب مستوى الطلب بناءً على:
- **العرض (30%)**: عدد المركبات المتاحة
- **الاهتمام (50%)**: متوسط المشاهدات اليومية
- **النشاط (20%)**: عدد الإعلانات الجديدة

```typescript
demandScore = (count * 0.3) + (avgViewsPerDay * 0.5) + (listings.length * 0.2)
demandPercentage = (demandScore / 100) * 100
```

#### 2. نظام الترتيب والميداليات
- 🥇 **المركز الأول**: ميدالية ذهبية
- 🥈 **المركز الثاني**: ميدالية فضية
- 🥉 **المركز الثالث**: ميدالية برونزية
- **المراكز الأخرى**: أرقام ملونة

#### 3. استخراج العلامات التجارية الشائعة
يتم تحليل البيانات لاستخراج أكثر 5 علامات تجارية شعبية في كل فئة

#### 4. مؤشر الطلب المرئي
شريط تقدم ديناميكي يعرض مستوى الطلب بنسبة مئوية

### الفئات المدعومة
1. **سيدان** (sedan) - سيارات عائلية مريحة
2. **دفع رباعي** (suv) - قوة وأداء
3. **هاتشباك** (hatchback) - مدمجة واقتصادية
4. **كوبيه** (coupe) - رياضية وأنيقة
5. **ستيشن واجن** (wagon) - مساحة واسعة
6. **كشف** (convertible) - أناقة وحرية
7. **بيك أب** (pickup) - قوة وعملية
8. **ميني فان** (minivan) - للعائلات الكبيرة

## 3. شاهدت مؤخراً | Recent Browsing Section

### الملف | File
`RecentBrowsingSection.tsx`

### الوصف | Description
قسم تتبع ذكي يحفظ سجل المشاهدات للمستخدم مع معلومات زمنية وإحصائية.

Smart tracking section that saves user's browsing history with temporal and statistical information.

### الميزات الذكية | Smart Features

#### 1. التخزين المحلي الذكي
- استخدام `localStorage` لحفظ السجل
- الحد الأقصى: 12 مركبة
- تحديث تلقائي عند كل مشاهدة

#### 2. تتبع عدد المشاهدات
- حساب عدد مرات مشاهدة كل مركبة
- عرض العدد في شارة مميزة

#### 3. الطوابع الزمنية الذكية
```typescript

  // أقل من يوم: "منذ X ساعة"
  // أقل من أسبوع: "منذ X يوم"
  // أكثر من أسبوع: التاريخ الكامل
}
```

#### 4. الترتيب التلقائي
يتم ترتيب المركبات حسب آخر مشاهدة (الأحدث أولاً)
