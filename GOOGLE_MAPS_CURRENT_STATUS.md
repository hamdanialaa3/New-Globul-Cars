## 🔄 لتفعيل Google Maps (اختياري)

إذا أردت استخدام Google Maps الحقيقية:

### الخطوات:

#### 1️⃣ افتح Google Cloud Console:
```
https://console.cloud.google.com/apis/credentials?project=fire-new-globul
```

#### 2️⃣ اضغط على API Key ثم Edit ✏️

#### 3️⃣ غيّر الإعدادات:
```
Application restrictions: None
API restrictions: Don't restrict key
```

#### 4️⃣ Save 💾

#### 5️⃣ في الكود، غيّر:
```tsx
// في src/pages/HomePage/CityCarsSection/index.tsx
// السطر 20:

const [mapError, setMapError] = useState(false); // ✅ استخدام Google Maps
```

#### 6️⃣ أعد تشغيل:
```bash
Ctrl + C
npm start
```
