# دليل إعادة تعيين إعدادات Cursor

## ✅ ما تم إنجازه

تم إعادة تعيين إعدادات Cursor إلى الإعدادات الأساسية لتحسين الأداء.

### التغييرات التي تمت:

1. **إنشاء نسخة احتياطية** من جميع الإعدادات الحالية
2. **إعادة تعيين `settings.json`** إلى إعدادات أساسية محسّنة
3. **إعادة تعيين `keybindings.json`** إلى القيم الافتراضية

## 📁 موقع الملفات

### الإعدادات الحالية:
- `%APPDATA%\Cursor\User\settings.json`
- `%APPDATA%\Cursor\User\keybindings.json`

### النسخة الاحتياطية:
- `%APPDATA%\Cursor\User\backup_YYYYMMDD_HHMMSS\`

## 🔄 كيفية استعادة الإعدادات القديمة

إذا أردت استعادة الإعدادات القديمة:

1. افتح مجلد النسخة الاحتياطية:
   ```
   %APPDATA%\Cursor\User\backup_*
   ```

2. انسخ الملفات من المجلد الاحتياطي:
   - `settings.json.backup` → `settings.json`
   - `keybindings.json.backup` → `keybindings.json`

3. أعد تشغيل Cursor

## ⚡ تحسينات الأداء المطبقة

الإعدادات الجديدة تشمل:

- ✅ تعطيل التحديثات التلقائية للإضافات
- ✅ تعطيل `git.autofetch` (يقلل من استهلاك الموارد)
- ✅ إعدادات بحث محسّنة (استثناء node_modules)
- ✅ إعدادات ملفات محسّنة

## 🚀 الخطوات التالية

1. **أغلق Cursor تماماً** (من Task Manager إذا لزم الأمر)
2. **أعد فتح Cursor**
3. **تحقق من الأداء** - يجب أن يكون أسرع الآن

## 📝 ملاحظات

- جميع الإعدادات المخصصة تم حفظها في النسخة الاحتياطية
- يمكنك إعادة تخصيص الإعدادات تدريجياً حسب الحاجة
- إذا واجهت أي مشاكل، استخدم النسخة الاحتياطية
