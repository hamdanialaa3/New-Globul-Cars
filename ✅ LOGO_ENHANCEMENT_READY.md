## 🎨 **التأثيرات التقنية:**

### **drop-shadow (الإطار الضوئي):**
```css
drop-shadow(0 0 2px rgba(3, 233, 244, 0.8))   /* طبقة 1 - رفيع جداً */
drop-shadow(0 0 4px rgba(3, 233, 244, 0.6))   /* طبقة 2 */
drop-shadow(0 0 8px rgba(3, 233, 244, 0.4))   /* طبقة 3 */
drop-shadow(0 0 12px rgba(3, 233, 244, 0.2))  /* طبقة 4 - وهج خارجي */
```

### **brightness + contrast (تحسين الجودة):**
```css
brightness(1.1)   /* أفتح بـ 10% */
contrast(1.15)    /* تباين أقوى بـ 15% */
saturate(1.3)     /* ألوان أكثر حيوية بـ 30% */
```
