═══════════════════════════════════════════════════════════════════════════════

🎯 الأنواع الثلاثة للبروفايل:
───────────────────────────────────────────────────────────────────────────────

1. Private Profile (بروفايل الأفراد)
   ├─ Plan Tiers: free, premium
   ├─ Max Listings: 3 (free), 10 (premium)
   ├─ Theme: Orange (#FF8F10)
   └─ Features: Basic listing, messaging, favorites

2. Dealer Profile (بروفايل الوكلاء)
   ├─ Plan Tiers: dealer_basic, dealer_pro, dealer_enterprise
   ├─ Max Listings: 50 (basic), 150 (pro), unlimited (enterprise)
   ├─ Theme: Green (#16a34a)
   ├─ Features: Analytics, campaigns, team management, bulk operations
   └─ Reference: dealershipRef → dealerships/{uid}

3. Company Profile (بروفايل الشركات)
   ├─ Plan Tiers: company_starter, company_pro, company_enterprise
   ├─ Max Listings: 100 (starter), unlimited (pro/enterprise)
   ├─ Theme: Blue (#1d4ed8)
   ├─ Features: Advanced analytics, API access, CSV import/export
   └─ Reference: companyRef → companies/{uid}
