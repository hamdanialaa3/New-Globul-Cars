## 📋 الشروط لكل نوع:

### 1. Private (Частно лице):
**الحقوق:**
- ✓ حد أقصى 3 إعلانات نشطة في نفس الوقت
- ✓ حتى 5 سيارات للبيع سنوياً
- ✓ سيارات شخصية فقط (ليس تجاري)
- ✓ بدون حاجة لتسجيل تجاري
- ✓ بدون التزامات ضريبة القيمة المضافة
- ✓ البيع يجب ألا يكون نشاطاً أساسياً

**التحذير:**
⚠️ إذا كنت تبيع أكثر من 5 سيارات سنوياً أو كان البيع نشاطاً أساسياً، يجب التسجيل كتاجر.

**المتطلبات:**
- بطاقة هوية صالحة أو جواز سفر
- مستندات ملكية السيارة
- رقم هاتف محمول صالح
- عنوان بريد إلكتروني صالح

---

### 2. Dealer (Автокъща / Търговец):
**الحقوق:**
- ✓ إعلانات غير محدودة
- ✓ التسجيل التجاري إلزامي
- ✓ رقم BULSTAT إلزامي
- ✓ تسجيل ضريبة القيمة المضافة (إذا كان الدخل > 50,000 ليف)
- ✓ تأمين المسؤولية إلزامي
- ✓ عنوان تجاري وساعات عمل
- ✓ ضمان على السيارات المباعة (12 شهر)

**التحذير:**
⚠️ مطلوب التسجيل في السجل التجاري والالتزام بقانون حماية المستهلك.

**المتطلبات:**
- التسجيل في السجل التجاري
- رقم BULSTAT
- رقم ضريبة القيمة المضافة (عند الحاجة)
- تأمين المسؤولية
- عنوان تجاري
- حساب بنكي للشركة
- مستندات ملكية السيارات

---

### 3. Company (Корпоративна компания):
**الحقوق:**
- ✓ إعلانات غير محدودة
- ✓ التسجيل كشركة تجارية (ООД/ЕООД/АД)
- ✓ رقم BULSTAT وضريبة القيمة المضافة إلزامي
- ✓ تأمين الشركات إلزامي
- ✓ تقارير محاسبية حسب قانون المحاسبة
- ✓ ضريبة دخل الشركات 10% على الدخل المحقق
- ✓ إمكانيات إدارة الفريق
- ✓ وصول API للتكاملات

**التحذير:**
⚠️ الالتزام الكامل بالقانون التجاري وقانون المحاسبة والتشريعات الضريبية لجمهورية بلغاريا.

**المتطلبات:**
- التسجيل في السجل التجاري
- رقم BULSTAT
- رقم ضريبة القيمة المضافة
- تأمين الشركات
- تدقيق وتقارير محاسبية
- العنوان القانوني للشركة
- ممثل الشركة
- حساب بنكي للشركات
