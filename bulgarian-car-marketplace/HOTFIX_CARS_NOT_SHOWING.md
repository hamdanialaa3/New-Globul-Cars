# 🔧 إصلاح عاجل: السيارات لا تظهر - Hotfix: Cars Not Showing

## 🐛 المشكلة
بعد تحديث الخدمات، السيارات المحفوظة لا تظهر في الصفحة.

## 🔍 السبب
الخدمة الموحدة كانت تبحث عن حقول `isActive` و `isSold` في Firestore، لكن السيارات القديمة لا تحتوي على هذه الحقول.

```typescript
// ❌ الكود القديم - يفشل مع البيانات القديمة
where('isActive', '==', true),
where('isSold', '==', false),
```

## ✅ الحل
تم تغيير الاستراتيجية:
1. جلب السيارات بدون فلاتر صارمة
2. الفلترة من جانب العميل (client-side)
3. التعامل مع الحقول المفقودة بقيم افتراضية

```typescript
// ✅ الكود الجديد - يعمل مع جميع البيانات
const isActive = car.isActive !== false; // افتراضي: true
const isSold = car.isSold === true;      // افتراضي: false
```

## 📝 التغييرات

### 1. getFeaturedCars()
- إزالة `where` filters من Firestore
- إضافة فلترة client-side
- جلب ضعف العدد للتعويض

### 2. searchCars()
- نقل فلاتر `isActive` و `isSold` إلى client-side
- إضافة قيم افتراضية للحقول المفقودة
- الحفاظ على باقي الفلاتر

## 🚀 النتيجة
- ✅ السيارات القديمة تظهر الآن
- ✅ السيارات الجديدة تعمل بشكل صحيح
- ✅ التوافق الكامل مع جميع البيانات
