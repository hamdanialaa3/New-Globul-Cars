## 🔧 التفاصيل التقنية

### 1️⃣ نقطة LED المشعة:

```css
&::before {
  width: 16px;
  height: 16px;
  border-radius: 50%;
  background: #FFFFFF; /* يتغير حسب اللون */
  box-shadow: 
    0 0 30px 10px currentColor,
    0 0 60px 20px currentColor,
    0 0 90px 30px rgba(255, 255, 255, 0.3);
  filter: brightness(2); /* إضاءة مضاعفة */
}
```

### 2️⃣ مسار الحركة (9 ثواني):

**التوقيت:**
- **0-33%** (3 ثواني): LED أبيض (يدور على الحافة العلوية)
- **33-66%** (3 ثواني): LED أخضر (يدور على الحافة اليمنى)
- **66-100%** (3 ثواني): LED أحمر (يدور على الحافة السفلية والعودة)

### 3️⃣ الإضاءة (Glow Effect):

**أبيض (White):**
```css
box-shadow: 
  0 0 30px 10px rgba(255, 255, 255, 1),     /* توهج داخلي */
  0 0 60px 20px rgba(255, 255, 255, 0.7),   /* توهج متوسط */
  0 0 90px 30px rgba(255, 255, 255, 0.4);   /* توهج خارجي */
```

**أخضر (Green):**
```css
background: #00966E;
box-shadow: 
  0 0 30px 10px rgba(0, 255, 136, 1),
  0 0 60px 20px rgba(0, 255, 136, 0.7),
  0 0 90px 30px rgba(0, 255, 136, 0.4);
```

**أحمر (Red):**
```css
background: #D62612;
box-shadow: 
  0 0 30px 10px rgba(255, 60, 40, 1),
  0 0 60px 20px rgba(255, 60, 40, 0.7),
  0 0 90px 30px rgba(255, 60, 40, 0.4);
```

## 🎬 الأنيميشن

### Animation 1: `bulgariaBorderLED` (المسار)

```css
@keyframes bulgariaBorderLED {
  /* 0-25%: Top edge (أبيض) */
  0%, 25% {
    top: -8px;
    left: calc(-8px + (100% + 16px) * progress);
  }
  
  /* 25-50%: Right edge (أخضر) */
  25.01%, 50% {
    top: calc(-8px + (100% + 16px) * progress);
    right: -8px;
  }
  
  /* 50-75%: Bottom edge (أحمر) */
  50.01%, 75% {
    bottom: -8px;
    right: calc(-8px + (100% + 16px) * progress);
  }
  
  /* 75-100%: Left edge (عودة) */
  75.01%, 100% {
    left: -8px;
    bottom: calc(-8px + (100% + 16px) * progress);
  }
}
```
