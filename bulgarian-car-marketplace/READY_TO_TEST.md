## ⚠️ **إذا ظهرت مشاكل:**

### **Error 1: Cannot find module 'browser-image-compression'**
```bash
npm install browser-image-compression --legacy-peer-deps
```

### **Error 2: Firebase Storage not enabled**
```
1. افتح: https://console.firebase.google.com/project/studio-448742006-a3493/storage
2. Click "Get Started"
3. Select "europe-west1"
4. Click "Done"
```

### **Error 3: Images not uploading**
```bash
# Deploy storage rules
firebase deploy --only storage

# Check in Firebase Console → Storage → Rules
```

### **Error 4: TypeScript errors**
```bash
# Check الأخطاء
npm run build

# أو شغّل بدون checks (مؤقتاً)
SKIP_PREFLIGHT_CHECK=true npm start
```
