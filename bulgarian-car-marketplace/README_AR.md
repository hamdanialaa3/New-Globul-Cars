## 🚀 البدء السريع

### المتطلبات:
```bash
Node.js >= 16.x
npm >= 8.x
```

### التثبيت:
```bash
# 1. استنساخ المشروع
git clone [repository-url]

# 2. تثبيت الحزم
npm install

# 3. تشغيل التطبيق
npm start
```

### الأوامر المفيدة:
```bash
npm start              # تشغيل التطوير
npm run build          # بناء للإنتاج
npm test               # تشغيل الاختبارات
npm run analyze        # تحليل الملفات
```
