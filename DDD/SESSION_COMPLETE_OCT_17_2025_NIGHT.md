## 📊 Analytics - ملخص

### Collections في Firebase:

```
analytics_events:
  • كل event
  • type: profile_view, car_view, inquiry, favorite
  • timestamp
  • visitor/user IDs

profile_metrics:
  • ملخص لكل user
  • profileViews, carViews, inquiries, etc.
  • responseTime average
  • lastUpdated
```

---

### Tracking Methods:

```
✅ trackProfileView()     → عند زيارة profile
✅ trackCarView()         → عند مشاهدة سيارة
✅ trackInquiry()         → عند إرسال رسالة
✅ trackFavorite()        → عند إضافة مفضلة
✅ trackResponse()        → عند الرد
```

---

### Reading Methods:

```
✅ getAnalytics(userId, period)
   → يجلب كل البيانات
   → يحسب الـ changes
   → يُنشئ viewsByDay
   → يحسب conversion rate
```

---

### Auto-Tracking:

```
✅ useProfileTracking(userId)
   → في ProfilePage
   → يتتبع تلقائياً بعد 2 ثانية
   → يتخطى own profile

✅ useCarViewTracking(carId, sellerId)
   → في CarDetailsPage
   → يتتبع تلقائياً بعد 3 ثواني
   → يتخطى own cars
```
