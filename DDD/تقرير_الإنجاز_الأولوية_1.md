### 2. نظام التعليقات الكامل (Comments System)
**الملف:** `src/services/social/comments.service.ts`

**الإمكانيات:**
- ✏️ إنشاء تعليق جديد مع معلومات الكاتب
- 📖 قراءة التعليقات مع ترقيم الصفحات (20 تعليق/صفحة)
- 🔁 الحصول على الردود المتداخلة
- ✏️ تعديل التعليق (المحتوى فقط)
- 🗑️ حذف التعليق (soft delete)
- ❤️ الإعجاب/إلغاء الإعجاب بالتعليق
- 🔢 عد إجمالي التعليقات للمنشور

**البيانات:**
```typescript
interface Comment {
  id: string;              // معرّف التعليق
  postId: string;          // معرّف المنشور
  authorId: string;        // معرّف الكاتب
  authorName: string;      // اسم الكاتب
  authorPhoto?: string;    // صورة الكاتب
  content: string;         // محتوى التعليق
  createdAt: Timestamp;    // تاريخ الإنشاء
  likes: string[];         // قائمة المعجبين
  likesCount: number;      // عدد الإعجابات
  parentCommentId?: string; // للردود المتداخلة
  isDeleted: boolean;      // علامة الحذف
}
```
