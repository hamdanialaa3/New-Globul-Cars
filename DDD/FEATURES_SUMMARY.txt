🗺️ GOOGLE MAPS - 7 APIs
=========================================

1. Maps JavaScript API ✅
   → خريطة بلغاريا التفاعلية
   → في الصفحة الرئيسية

2. Geocoding API ✅
   → تحويل المدن لإحداثيات
   → في جميع الصفحات

3. Places API (New) ✅
   → بحث ذكي عن المدن
   → اقتراحات فورية

4. Distance Matrix API ✅
   → حساب المسافة والوقت
   → "15.3 km - 25 min"

5. Directions API ✅
   → زر "Get Directions"
   → يفتح Google Maps

6. Time Zone API ✅
   → الوقت المحلي للبائع
   → "14:30"

7. Maps Embed API ✅
   → خريطة ثابتة مدمجة
   → في صفحة التفاصيل
