## 🎯 الآلية الكاملة

### كيف يعمل الربط:

```
Step 1: المستخدم يضيف سيارة
├─ يختار المدينة: "صوفيا"
├─ sellWorkflowService يحفظ:
│  location: {
│    cityId: 'sofia-city',
│    cityNameBg: 'София',
│    coordinates: { lat: 42.6977, lng: 23.3219 }
│  }
└─ ✅ محفوظة في Firestore

Step 2: الصفحة الرئيسية تُفتح
├─ CityCarsSection يحمّل
├─ CityCarCountService.getAllCityCounts()
├─ يبحث: where('location.cityId', '==', 'sofia-city')
├─ يجد: 25 سيارة
└─ ✅ يعرض: صوفيا (25)

Step 3: عرض على الخريطة
├─ LeafletBulgariaMap
├─ يستلم: { 'sofia': 25, 'plovdiv': 15, ... }
├─ يعرض نقطة زرقاء على صوفيا
├─ يكتب: 25 بجانبها
└─ ✅ مرتبط!

Step 4: المستخدم ينقر على صوفيا
├─ navigate('/cars?city=sofia-city')
├─ CarsPage يفلتر: where('location.cityId', '==', 'sofia-city')
├─ يعرض: 25 سيارة من صوفيا فقط
└─ ✅ كل شيء مرتبط!
```
