```javascript
// مثال على البنية
const carBrands = {
  "mercedes-benz": {
    name: "Mercedes-Benz",
    nameAr: "مرسيدس-بنز",
    country: "Germany",
    seriesCount: 37,
    category: "luxury",
    series: ["A-Class", "C-Class", "E-Class", ...]
  },
  // ... باقي العلامات
}
```

### 2. فئات البحث

- البحث حسب العلامة التجارية
- البحث حسب نوع الهيكل (SUV, Sedan, etc.)
- البحث حسب نوع الوقود (بنزين, كهربائي, هجين)
- البحث حسب الفئة (فاخرة, عادية, رياضية)
- البحث حسب البلد المصنع

### 3. الفلاتر المقترحة

```json
{
  "brand": ["Mercedes-Benz", "BMW", "Audi", ...],
  "bodyType": ["Sedan", "SUV", "Hatchback", ...],
  "fuelType": ["Petrol", "Diesel", "Electric", "Hybrid"],
  "transmission": ["Manual", "Automatic"],
  "priceRange": [min, max],
  "year": [min, max],
  "mileage": [min, max]
}
```

### 4. الصفحات المقترحة

- صفحة لكل علامة تجارية
- صفحة لكل سلسلة
- صفحة مقارنة بين السلاسل
- صفحة البحث المتقدم
- صفحة السيارات الكهربائية
- صفحة السيارات الفاخرة
