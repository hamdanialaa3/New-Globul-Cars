## 🔥 **الأخطاء المحتملة وتفسيرها:**

### **1. auth/unauthorized-domain**
```
❌ المشكلة: النطاق غير مسموح به
✅ الحل: أضف النطاق في Firebase → Authentication → Settings → Authorized domains
```

### **2. auth/popup-blocked**
```
❌ المشكلة: المتصفح حظر النافذة المنبثقة
✅ الحل: اسمح بالنوافذ المنبثقة أو جرب Redirect method
```

### **3. auth/operation-not-allowed**
```
❌ المشكلة: Google Sign-In غير مفعل
✅ الحل: فعّل Google في Firebase Console → Authentication → Sign-in method
```

### **4. auth/internal-error**
```
❌ المشكلة: OAuth settings غير صحيحة في Google Cloud
✅ الحل: تحقق من Authorized JavaScript origins و Authorized redirect URIs
```
