## ⚠️ **إذا ظهرت أخطاء:**

### **Error 1: Module not found 'browser-image-compression'**
```bash
cd bulgarian-car-marketplace
npm install browser-image-compression --legacy-peer-deps
```

### **Error 2: Firebase Storage not enabled**
```
□ Go to Firebase Console
□ Storage → Get Started
□ Select: europe-west1
□ Click Done
```

### **Error 3: TypeScript errors**
```bash
# Check linting
npm run lint

# أو تجاهل مؤقتاً
# في tsconfig.json أضف: "skipLibCheck": true
```
