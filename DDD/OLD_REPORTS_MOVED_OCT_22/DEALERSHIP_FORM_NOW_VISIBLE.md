## 🔄 كيف تجربه الآن | How to Test NOW

### الخطوة 1: تشغيل المشروع
```bash
cd bulgarian-car-marketplace
npm start
```

### الخطوة 2: تسجيل الدخول
افتح المتصفح: http://localhost:3000

### الخطوة 3: الذهاب لصفحة Profile
انقر على الصورة الشخصية أو Profile

### الخطوة 4: تبديل نوع الحساب
1. ابحث عن أزرار نوع الحساب (أعلى الصفحة تحت الصورة)
2. انقر على زر **"Dealer"** (أخضر)
3. أكّد في النافذة المنبثقة

### الخطوة 5: اذهب لتبويب Settings
انقر على تبويب **"Settings"** / **"Настройки"**

### الخطوة 6: املأ النموذج! 🎉
- ستجد **Privacy Settings** في الأعلى
- ستجد **Dealership Information Form** في الأسفل
- املأ الحقول المطلوبة (*)
- اختر الخدمات
- انقر **Save**
- تهانينا! 🎊
