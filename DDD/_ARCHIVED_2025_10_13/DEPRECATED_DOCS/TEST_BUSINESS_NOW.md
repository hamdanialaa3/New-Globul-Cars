# 🧪 اختبر البروفايل التجاري الآن!

## ⚡ **اختبار سريع (5 دقائق)**

---

## 📍 **الخطوة 1: افتح البروفايل**

```bash
http://localhost:3000/profile
```

✅ **المتوقع:**
- البروفايل يظهر عادي (خلفية بيضاء)
- بطاقة زرقاء في الجانب: "Upgrade to Business"

---

## 📍 **الخطوة 2: ترقية لتجاري**

```bash
1. اضغط على البطاقة الزرقاء في الجانب
   أو
   اضغط "Edit Profile" ثم "🏢 Business"
```

✅ **المتوقع:**
- الصفحة تتحرك للأعلى
- وضع التحرير يفتح
- زر "Business" محدد (برتقالي)
- رسالة تحذير صفراء تظهر

```bash
املأ على الأقل:

✓ نوع العمل: اختر "🚗 Автосалон"

```

## 🎉 **الخطوة 5: شاهد السحر!**

### **يجب أن ترى:**

```
✅ 1. خلفية السيارات (مضببة)
      └─> صورة معرض سيارات خلف كل شيء

✅ 2. شريط LED أزرق في الأعلى
      └─> متحرك من اليسار لليمين

✅ 3. شريط LED أزرق في الأسفل
      └─> متحرك من اليمين لليسار

✅ 4. شارة "BUSINESS ACCOUNT" أعلى اليمين
      └─> مع أيقونة 🏢

✅ 5. البطاقات أصبحت زجاجية (شفافة قليلاً)
      └─> يمكنك رؤية الخلفية من خلالها

✅ 6. لون أزرق احترافي في كل مكان
```

## 🔍 **إذا لم تشاهد التغيير:**

### **افتح Console (F12):**

```bash
1. اضغط F12
2. انتقل لـ Console
3. ابحث عن:
   🏢 BusinessBackground - isBusinessAccount: true

إذا رأيت:
✅ true  → الكود يعمل! تحقق من الصور
❌ false → accountType لم يُحفظ، أعد الخطوة 4
```

---

### **تحقق من الصور:**

```bash
1. F12 → Network tab
2. Reload page (F5)
3. Filter: "Pic"
4. يجب أن ترى:
   ✅ pexels-boris-dahm...jpg (Status: 200)
   ✅ pexels-bylukemiller...jpg (Status: 200)
   
إذا رأيت:
❌ 404 → الصور غير موجودة، أعد نسخها
```

---

### **تحقق من Firestore:**

```bash
1. افتح Firebase Console
2. Firestore Database
3. users → {your-uid}
4. يجب أن ترى:
   ✅ accountType: "business"
   ✅ businessName: "..."
   ✅ businessType: "dealership"
```

## 🐛 **حل سريع للمشاكل:**

### **المشكلة: لا يظهر شيء**

```bash
الحل:
1. افتح Console (F12)
2. أعد تحميل الصفحة (F5)
3. اقرأ رسائل الـ console
4. إذا رأيت أخطاء → أبلغني بها
5. إذا رأيت "false" → أعد الحفظ
```

---

### **المشكلة: الصور لا تظهر**

```bash
الحل:
1. تأكد من تشغيل npm start
2. تحقق من Network tab (404?)
3. تأكد من وجود الصور في:
   public/assets/images/Pic/
```

---

### **المشكلة: LED لا يتحرك**

```bash
الحل:
1. تأكد من المتصفح حديث (Chrome/Edge)
2. تحقق من دعم CSS animations
3. Inspect element → check animation-name
```
