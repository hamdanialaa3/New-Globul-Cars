Bulgarian Car Marketplace - Development Server

⚡ الطريقة الأسهل والأسرع (مضمونة 100%)

الخطوة 1: افتح ملف START_SERVER_FIXED.bat

انقر نقراً مزدوجاً على الملف:

bulgarian-car-marketplace/START_SERVER_FIXED.bat

✅ سيقوم الملف تلقائياً بـ:

  • فحص Node.js و npm
  • تثبيت المكتبات إذا لزم الأمر
  • فحص ملف .env
  • تشغيل الخادم على المنفذ 3000
  • فتح المتصفح تلقائياً
الخطوة 2: انتظر رسالة النجاح

عند رؤية هذه الرسالة:

Compiled successfully! You can now view bulgarian-car-marketplace in the browser. Local: http://localhost:3000 On Your Network: http://192.168.x.x:3000

✅ الخادم يعمل بنجاح!

الخطوة 3: افتح المتصفح

إذا لم يفتح المتصفح تلقائياً، افتح يدوياً:

http://localhost:3000

🔧 الطريقة اليدوية (للمطورين)

الخطوة 1: افتح PowerShell أو CMD

اضغط Win + R ثم اكتب:

powershell
الخطوة 2: انتقل إلى المجلد
cd "c:\Users\hamda\Desktop\New Globul Cars\bulgarian-car-marketplace"
الخطوة 3: تأكد من تثبيت المكتبات
npm install --legacy-peer-deps

⏱️ قد يستغرق 3-5 دقائق في المرة الأولى

الخطوة 4: شغّل الخادم
npm start

⚠️ إذا واجهتك مشكلة "Cache" في المتصفح

جرّب أحد الحلول التالية:

1️⃣ Hard Refresh (الأسرع)

اضغط: Ctrl + Shift + R

أو: Ctrl + F5

2️⃣ Clear Cache من DevTools

  • اضغط F12 لفتح DevTools
  • انقر بزر الماوس الأيمن على زر Refresh 🔄
  • اختر: "Empty Cache and Hard Reload"

3️⃣ Incognito Mode (مضمون 100%)

اضغط: Ctrl + Shift + N

ثم افتح: http://localhost:3000

❌ حل المشاكل الشائعة

مشكلة: "npm is not recognized"

✅ الحل: ثبّت Node.js من https://nodejs.org/

مشكلة: "Port 3000 already in use"

✅ الحل: أوقف الخادم القديم بالضغط على Ctrl + C

مشكلة: "Module not found"

✅ الحل: احذف مجلد node_modules ثم:

npm install --legacy-peer-deps

مشكلة: الصفحة بيضاء فارغة

✅ الحل: افتح DevTools بالضغط على F12 وافحص Console للأخطاء

✅ التحقق من نجاح التشغيل

يجب أن ترى في Terminal:

webpack compiled successfully Compiled successfully! You can now view bulgarian-car-marketplace in the browser. Local: http://localhost:3000
