## 🔄 خطة الهجرة التدريجية

### الطريقة: **Incremental Migration** (هجرة تدريجية)

#### الخطوة 1: إنشاء Workspace Structure
```bash
# إنشاء packages directory
mkdir packages
cd packages

# إنشاء packages الأساسية
mkdir core ui services app
```

#### الخطوة 2: نقل Core أولاً
- نقل `contexts/` → `packages/core/src/contexts/`
- نقل `types/` → `packages/core/src/types/`
- نقل `utils/` → `packages/core/src/utils/`
- نقل `constants/` → `packages/core/src/constants/`
- نقل `config/` → `packages/core/src/config/`

#### الخطوة 3: تحديث Imports
```typescript
// قبل
import { useAuth } from '@/contexts/AuthContext';

// بعد
import { useAuth } from '@globul-cars/core/contexts';
```

#### الخطوة 4: نقل UI Components
- نقل `components/Header/` → `packages/ui/src/components/Header/`
- نقل `components/Footer/` → `packages/ui/src/components/Footer/`
- نقل `styles/` → `packages/ui/src/styles/`

#### الخطوة 5: نقل Services
- نقل `services/` → `packages/services/src/`
- تنظيم حسب الوظيفة

#### الخطوة 6: نقل الوحدات الوظيفية
- نقل `pages/02_authentication/` → `packages/auth/src/pages/`
- نقل `pages/04_car-selling/` → `packages/cars/src/pages/`
- نقل `pages/03_user-pages/profile/` → `packages/profile/src/pages/`
- وهكذا...

#### الخطوة 7: تحديث App Shell
- تحديث `App.tsx` لاستخدام packages
