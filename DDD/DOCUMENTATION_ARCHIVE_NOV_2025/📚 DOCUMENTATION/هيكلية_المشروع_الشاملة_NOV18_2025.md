## 📊 نظرة عامة (Overview)

### نوع المشروع
**Monorepo** - منصة سوق سيارات بلغارية متكاملة مع:
- ✅ Frontend (React 19 SPA)
- ✅ Backend (Firebase Cloud Functions)
- ✅ AI Services (Python ML Model)
- ✅ Infrastructure (Firebase, AWS, Google Cloud)

### التقنيات الرئيسية
- **Frontend:** React 19, TypeScript, Styled Components
- **Backend:** Firebase Functions (Node.js/TypeScript)
- **Database:** Firestore, Real-time Database
- **Storage:** Firebase Storage, AWS S3
- **AI/ML:** Python (XGBoost), Google Vertex AI, Gemini
- **Search:** Algolia
- **Maps:** Google Maps, Leaflet
- **Payments:** Stripe
- **IoT:** AWS IoT Core
