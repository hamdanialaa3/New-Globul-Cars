# AdvancedFilterSystemMobile

نظام التصفية المتقدم للأجهزة المحمولة في منصة Globul Cars.

## الهيكل المعماري

```
AdvancedFilterSystemMobile/
├── AdvancedFilterSystemMobile.tsx  # المكون الرئيسي
├── types.ts                        # تعريفات الأنواع
├── styles.ts                       # مكونات التصميم
├── hooks/
│   └── useAdvancedFilter.ts        # خطاف إدارة الحالة
├── index.ts                        # نقطة التصدير
└── README.md                       # التوثيق
```

## المكونات

### AdvancedFilterSystemMobile.tsx
المكون الرئيسي المسؤول عن عرض واجهة التصفية المتقدمة. يستخدم:
- `useAdvancedFilter` لإدارة الحالة والمنطق
- مكونات التصميم من `styles.ts`
- أنواع البيانات من `types.ts`

### useAdvancedFilter Hook
خطاف مخصص لإدارة:
- حالة الفلاتر والأقسام الموسعة
- تحديث الفلاتر وإعادة تعيينها
- تبديل حالة الأقسام (موسع/مطوي)

### Types
تعريفات الأنواع:
- `FilterValue`: قيم الفلاتر
- `AdvancedFilterSystemMobileProps`: خصائص المكون
- `SectionState`: حالة الأقسام

### Styles
مكونات التصميم المستجيبة:
- `FilterContainer`: الحاوي الرئيسي
- `FilterHeader`: رأس نظام التصفية
- `FilterSections`: حاوي الأقسام
- `FilterSection`: قسم تصفية فردي
- `FilterGrid`: شبكة الفلاتر
- `FilterGroup`: مجموعة فلتر
- `FilterLabel`: تسمية الفلتر
- `FilterSelect`: قائمة منسدلة
- `FilterButton`: أزرار التصفية

## الاستخدام

```tsx
import AdvancedFilterSystemMobile from './components/AdvancedFilterSystemMobile';

const handleSearch = (filters) => {
  console.log('البحث بالفلاتر:', filters);
  // تنفيذ منطق البحث
};

const handleReset = () => {
  console.log('إعادة تعيين الفلاتر');
};

<AdvancedFilterSystemMobile
  onSearch={handleSearch}
  onReset={handleReset}
  onSaveSearch={(filters) => console.log('حفظ البحث:', filters)}
  loading={false}
/>
```

- **البيانات الأساسية**: ماركة السيارة، موديل، جيل، نوع الهيكل
