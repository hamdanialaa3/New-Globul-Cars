## المنصات المدعومة

### 1. Google Shopping

**رابط الـ Feed:**
```
سيكون بعد نشر Functions:
https://us-central1-fire-new-globul.cloudfunctions.net/googleMerchantFeed
```

**المميزات:**
- ظهور في نتائج بحث Google
- تبويب Shopping في Google
- إعلانات Shopping المجانية
- إعلانات Shopping المدفوعة

**خطوات الربط:**

1. افتح Google Merchant Center
   ```
   https://merchants.google.com
   ```

2. أنشئ حساب:
   - اسم النشاط: Globul Cars
   - البلد: Bulgaria
   - الموقع: https://mobilebg.eu

3. تحقق من ملكية الموقع:
   - أضف meta tag في موقعك
   - أو ارفع ملف HTML

4. أضف Product Feed:
   - Products -> Feeds -> Add Feed
   - Input method: Scheduled fetch
   - Feed URL: (الرابط أعلاه)
   - Fetch schedule: Hourly
   - احفظ

5. انتظر 24-48 ساعة للمراجعة

6. ستبدأ السيارات بالظهور في Google Shopping!

---

### 2. Instagram Shopping

**رابط الـ Feed:**
```
سيكون بعد نشر Functions:
https://us-central1-fire-new-globul.cloudfunctions.net/instagramShoppingFeed
```

**المميزات:**
- وسم المنتجات في المنشورات
- متجر Instagram
- Shopping في Stories
- Checkout مباشر

**خطوات الربط:**

1. حوّل حساب Instagram إلى Business Account

2. اربط مع Facebook Business Manager

3. في Facebook Commerce Manager:
   ```
   https://business.facebook.com/commerce
   ```

4. أنشئ Product Catalog:
   - نوع: Vehicles
   - اسم: Globul Cars Bulgaria

5. أضف Data Source:
   - Type: Data Feed
   - Feed URL: (الرابط أعلاه)
   - Schedule: Hourly

6. في Instagram:
   - Settings -> Business -> Shopping
   - اختر الكتالوج
   - قدّم للمراجعة

7. انتظر 1-3 أيام للموافقة

8. ستظهر السيارات في Instagram!

## الربط التلقائي - كيف يعمل؟

### عندما يضيف مستخدم سيارة:

```
المستخدم -> نموذج البيع
    |
    v
يملأ البيانات (Make, Model, Year, Price, Images, etc.)
    |
    v
يضغط "نشر الإعلان"
    |
    v
تُحفظ في Firestore مع status: 'active'
    |
    v  
خلال ساعة (تلقائياً):
    |
    +-> Google يقرأ Feed -> يضيف السيارة
    +-> Instagram يقرأ Feed -> يضيف السيارة
    +-> TikTok يقرأ Feed -> يضيف السيارة
    |
    v
السيارة موجودة في 4 أماكن:
1. موقعك (mobilebg.eu)
2. Google Shopping
3. Instagram Shop
4. TikTok Shop
```

### عندما تُباع السيارة:

```
المستخدم يضغط "تم البيع"
    |
    v
status تتغير إلى 'sold'
    |
    v
خلال ساعة (تلقائياً):
    |
    +-> Google يقرأ Feed -> يحذف السيارة
    +-> Instagram يقرأ Feed -> يحذف السيارة
    +-> TikTok يقرأ Feed -> يحذف السيارة
    |
    v
السيارة مخفية من كل المنصات!
```

**كل شيء تلقائي 100%**

## نشر Functions (مطلوب)

حالياً Functions غير منشورة بسبب مشكلة صلاحيات. الحل:

### الخيار 1: في Firebase Console

1. افتح:
   ```
   https://console.firebase.google.com/project/fire-new-globul/settings/iam
   ```

2. تأكد من وجود صلاحيات:
   - Service Account Token Creator
   - Cloud Run Invoker
   - Eventarc Event Receiver

3. إذا لم تكن موجودة، أضفها

4. ثم انشر:
   ```bash
   firebase deploy --only functions
   ```

### الخيار 2: عبر gcloud CLI

```bash
gcloud projects add-iam-policy-binding fire-new-globul \
  --member=serviceAccount:service-973379297533@gcp-sa-pubsub.iam.gserviceaccount.com \
  --role=roles/iam.serviceAccountTokenCreator

gcloud projects add-iam-policy-binding fire-new-globul \
  --member=serviceAccount:973379297533-compute@developer.gserviceaccount.com \
  --role=roles/run.invoker
```
