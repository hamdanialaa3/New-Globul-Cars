## 🚨 Red Flags - علامات التحذير

### توقف فوراً إذا رأيت:

```
🔴 TypeScript errors > 5
🔴 Build failed
🔴 Runtime errors في Console
🔴 Bundle size زاد > 20%
🔴 Build time زاد > 50%
🔴 Git diff ضخم جداً (> 100 files لمرحلة واحدة)
🔴 عدد ملفات غير متوقع
🔴 Merge conflicts كثيرة
```

**الإجراء:**
1. STOP - لا تستمر
2. راجع RISK_ANALYSIS_ENHANCED.md
3. طبق الحل المناسب
4. استمر فقط بعد حل المشكلة
