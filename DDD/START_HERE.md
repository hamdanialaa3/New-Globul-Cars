## 🎯 البدء السريع

### 1. التثبيت:
```bash
cd bulgarian-car-marketplace
npm install
```

### 2. إعداد Environment Variables:
```bash
# انسخ الملف النموذجي
cp .env.example .env

# افتح .env وأضف القيم الفعلية من Firebase
```

### 3. البدء:
```bash
npm start
```

**افتح:** `http://localhost:3000`
