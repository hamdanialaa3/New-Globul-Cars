## 🎮 طرق التشغيل

### 1️⃣ التشغيل الفوري
```batch
Auto-Start-Download.bat
```
- يبدأ فوراً مع إعدادات افتراضية (8 صور/علامة)
- مناسب للاستخدام السريع

### 2️⃣ التحكم الكامل
```batch
Ultimate-Download-Command.bat
```
- قائمة خيارات تفاعلية
- تحكم كامل في حجم التحميل

### 3️⃣ من سطر الأوامر
```bash
# تحميل مخصص
node -e "import('./continuous-bulk-downloader.js').then(async (m) => { const d = new m.default(); await d.quickStart(5); })"

# تشغيل مباشر
node continuous-bulk-downloader.js
```

### 4️⃣ برمجياً
```javascript
import ContinuousBulkDownloader from './continuous-bulk-downloader.js';

const downloader = new ContinuousBulkDownloader();

// تحميل مخصص
await downloader.quickStart(10);

// تحميل شامل كامل
await downloader.downloadEverything(15);
```

## 🚨 ملاحظات مهمة

### ⚠️ قبل البدء
- **تأكد من اتصال الإنترنت** المستقر
- **ستحتاج مساحة تخزين** (كل 100 صورة ≈ 5-8 MB)
- **التحميل مستمر** - لا توقف النافذة

### 💾 متطلبات المساحة
- اختبار سريع (132 صورة): ~7 MB
- متوسط (264 صورة): ~15 MB  
- كبير (440 صورة): ~25 MB
- ضخم (660 صورة): ~35 MB
- ضخم جداً (880 صورة): ~50 MB

### 🔄 إعادة التشغيل
- النظام **لا يحذف الصور الموجودة**
- سيضيف صور جديدة إلى الصور الحالية
- الأسماء فريدة تلقائياً (رقم تسلسلي)
