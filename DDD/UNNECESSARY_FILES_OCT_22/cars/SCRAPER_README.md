# NetCarShow Comprehensive Car Data Scraper

هذه الخوارزمية الشاملة لسحب جميع بيانات السيارات من موقع NetCarShow.com بدون استثناء أو تجاهل أي معلومات.

## الميزات

- **سحب شامل**: تستخرج جميع العلامات التجارية والموديلات والأجيال
- **تحديث تلقائي**: تحديث الملفات الموجودة بالبيانات الجديدة
- **معالجة الأخطاء**: معالجة شاملة للأخطاء وإعادة المحاولة
- **تنسيق منظم**: حفظ البيانات بتنسيق منظم وقابل للقراءة
- **تأخير احترافي**: تأخير بين الطلبات لتجنب حظر IP

## المتطلبات

- Node.js >= 18.0.0
- npm >= 8.0.0
- اتصال إنترنت مستقر

## التثبيت

```bash
npm install cheerio
```

## الاستخدام

### سحب جميع البيانات من البداية

```bash
node scraper.js scrape
```

### تحديث الملفات الموجودة

```bash
node scraper.js update
```

## الملفات المُنشأة/المحدثة

- `ABT.txt` - طرازات ABT
- `AC Schnitzer.txt` - طرازات AC Schnitzer
- `Acura.txt` - طرازات Acura
- `Alfa Romeo.txt` - طرازات Alfa Romeo
- `Alpina.txt` - طرازات Alpina
- `Apex.txt` - طرازات Apex
- وجميع العلامات التجارية الأخرى من NetCarShow

## الخوارزمية التفصيلية

### 1. استخراج العلامات التجارية
- يتم البحث عن جميع روابط العلامات في الصفحة الرئيسية
- استخراج الروابط الفريدة لكل علامة تجارية

### 2. استخراج الموديلات لكل علامة
- زيارة صفحة كل علامة تجارية
- استخراج جميع الموديلات والأجيال
- استخراج سنة الإصدار من اسم الموديل

### 3. استخراج التفاصيل الفنية (اختياري)
- زيارة صفحة كل موديل
- استخراج المواصفات الفنية (المحرك، القوة، الوزن، إلخ)

### 4. حفظ البيانات
- ترتيب الموديلات حسب السنة (الأحدث أولاً)
- حفظ في ملفات نصية منفصلة لكل علامة
- تنسيق واضح وقابل للقراءة

## معالجة الأخطاء

- **إعادة المحاولة**: إعادة المحاولة 3 مرات لكل طلب فاشل
- **تأخير تصاعدي**: زيادة التأخير عند فشل الطلبات
- **تسجيل الأخطاء**: تسجيل جميع الأخطاء للمراجعة
- **استمرارية**: الاستمرار في المعالجة حتى مع فشل بعض العلامات

## الأداء

- **تأخير احترافي**: 1 ثانية بين كل علامة تجارية
- **معالجة بالتسلسل**: معالجة علامة تلو الأخرى لتجنب التحميل الزائد
- **ذاكرة فعالة**: معالجة البيانات على دفعات صغيرة

## ملاحظات مهمة

- **الاحترام**: الخوارزمية مصممة لتكون محترمة للموقع
- **القانونية**: تأكد من الامتثال لشروط استخدام الموقع
- **التحديث**: يمكن تشغيل الخوارزمية مرة أخرى للحصول على البيانات الجديدة
- **النسخ الاحتياطي**: احتفظ بنسخة احتياطية من الملفات قبل التشغيل
