# 🚗 NetCarShow.com - نظام سحب الصور الشامل المتقدم

## 📋 نظرة عامة

هذا النظام المتطور مصمم لسحب **جميع صور السيارات** من موقع NetCarShow.com بشكل تلقائي وتنظيمها في المجلدات المناسبة حسب العلامة التجارية مع إعادة تسمية ذكية للصور.

## 🎯 المميزات الرئيسية

### 🔄 سحب شامل ومتقدم
- **استخراج جميع العلامات التجارية** تلقائياً من الموقع
- **اكتشاف جميع الموديلات** لكل علامة تجارية
- **استخراج متعدد المصادر** للصور:
  - صور IMG العادية
  - صور CSS Background
  - صور من JavaScript
  - بيانات JSON-LD المنظمة

### 📁 تنظيم تلقائي ذكي
- **توزيع تلقائي** للصور في مجلدات العلامات التجارية المناسبة
- **إعادة تسمية ذكية**: `العلامة_الموديل_الرقم_الطابع الزمني.jpg`
- **تجنب التكرار** - لا يُحمل نفس الصورة مرتين
- **تخطي الملفات الموجودة** لتوفير الوقت والبيانات

### 🛡️ حماية وأمان متقدمة
- **نظام إعادة المحاولة** مع Exponential Backoff
- **دوران User Agents** لتجنب الحظر
- **تأخير ذكي** بين الطلبات
- **معالجة شاملة للأخطاء**

### 📊 مراقبة وإحصائيات
- **إحصائيات مفصلة** في الوقت الفعلي
- **عرض التقدم** كل 10 دقائق
- **معلومات الأداء**: سرعة التحميل، حجم البيانات، إلخ
- **تسجيل شامل** لجميع العمليات

### 🎨 فلترة ذكية للصور
- **تجنب الصور الصغيرة** (أقل من 300x200)
- **استبعاد الأيقونات والشعارات**
- **قبول الجودة العالية** فقط
- **تحقق من صحة نوع المحتوى**

## 🛡️ معالجة الأخطاء

### الأخطاء الشائعة والحلول

#### 1. خطأ "ENOTFOUND" أو انقطاع الاتصال
```
✅ الحل: النظام يعيد المحاولة تلقائياً
⚠️  إذا استمر: تحقق من اتصال الإنترنت
```

#### 2. خطأ "Directory not found"
```
❌ السبب: مجلد brand_directories غير موجود
✅ الحل: تأكد من وجود المجلد مع مجلدات العلامات التجارية
```

#### 3. خطأ "Permission denied"
```
❌ السبب: عدم وجود صلاحيات كتابة
✅ الحل: تشغيل Command Prompt كمدير
```

#### 4. خطأ "Out of memory"
```
❌ السبب: استهلاك عالي للذاكرة
✅ الحل: إعادة تشغيل البرنامج - سيتخطى الملفات المُحملة
```

## 📈 تحسين الأداء

### نصائح لتحسين السرعة
1. **اتصال إنترنت سريع**: كلما كان أسرع، كان التحميل أسرع
2. **SSD**: استخدم قرص صلب من نوع SSD للحفظ
3. **RAM كافية**: 8GB+ مُفضلة
4. **إغلاق البرامج الأخرى**: لتوفير موارد النظام

### نصائح لتوفير البيانات
1. **شغل في الليل**: عادة البيانات أرخص
2. **اتصال لا محدود**: مُفضل بشدة
3. **مراقبة الاستهلاك**: قد يصل إلى عدة GB
