# القسم 30: Content Moderation and Prohibited Content (P1)

## 30.1 Content Policy

```
PROHIBITED LISTINGS:
1. Stolen Vehicles
2. Salvage/Total Loss (without disclosure)
3. Cloned/Modified VINs
4. Fraudulent Pricing (50% below market)
5. Inappropriate Images
6. Spam/Duplicate Listings
7. Non-Car Items

MODERATION WORKFLOW:

Automated (AI/Rules):
  - Price anomaly detection
  - Image content scanning
  - Duplicate detection
  - Prohibited keywords

Manual Review Queue:
  - Flagged listings in admin dashboard
  - User reports: reviewed within 24 hours

Actions:
  - Warning: email user
  - Suspension: listing hidden
  - Ban: account suspended
  - All logged with reason

IMAGE GUIDELINES:
  - Minimum: 3 images
  - Maximum: 20 images
  - Size: <5MB each
  - Required: exterior, interior, dashboard

TEXT GUIDELINES:
  - Minimum: 50 characters
  - No external links in description
  - Language: Bulgarian or English
  - Profanity filter: automatic

MODERATION SLAs:
  - User-reported: <24 hours
  - High-priority flags: <2 hours
  - First-time dealer: <48 hours
```
