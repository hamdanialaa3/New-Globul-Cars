# 🎬 خطوات بسيطة جداً - بالأرقام

## 🔢 **اتبع هذه الأرقام بالترتيب**

---

### 1️⃣ **افتح حسابك**
```
https://globul-cars-bg.app.n8n.cloud
```
سجل الدخول

---

### 2️⃣ **اضغط الزر الأزرق**
```
+ New Workflow
```

---

### 3️⃣ **ستفتح صفحة جديدة - ابحث عن الأيقونة**
```
⋯ (ثلاث نقاط في الشريط العلوي)
```

---

### 4️⃣ **اضغط الثلاث نقاط واختر**
```
Import from JSON
```

---

### 5️⃣ **ستفتح نافذة - اتركها مفتوحة وافتح الملف**
```
IMPORT_WORKFLOWS_GUIDE.md
```

---

### 6️⃣ **في الملف، ابحث عن النص الذي يبدأ بـ**
```json
{
  "name": "Globul Cars - Sell Process Started",
```

---

### 7️⃣ **انسخ كل شيء من أول `{` إلى آخر `}`**
تأكد أنك نسخت الكود كاملاً

---

### 8️⃣ **ارجع لنافذة n8n والصق النص**
اضغط Ctrl+V في المربع

---

### 9️⃣ **اضغط زر Import**
ستظهر صناديق مربوطة ببعض

---

### 🔟 **اضغط Save في الأعلى**
ثم اضغط Active (سيصبح أخضر)

---
