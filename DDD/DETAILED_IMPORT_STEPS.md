# 📖 دليل الاستيراد المُفصل خطوة بخطوة

## 🎯 **الهدف**: استيراد 2 workflows إلى حسابك في n8n

---

## 🚀 **الخطوة الأولى: فتح حسابك**

### 1. افتح المتصفح واذهب إلى:
```
https://globul-cars-bg.app.n8n.cloud
```

### 2. سجل الدخول بحسابك
- أدخل الإيميل وكلمة المرور الخاصة بك

---

## 📋 **الخطوة الثانية: إنشاء Workflow جديد**

### 1. ستجد في الصفحة الرئيسية زر اسمه:
```
+ New Workflow
```
**اضغط عليه**

### 2. ستفتح صفحة جديدة (محرر n8n) بها:
- منطقة فارغة في الوسط
- شريط علوي به أزرار
- قائمة جانبية بالـ nodes

---

## 📥 **الخطوة الثالثة: استيراد JSON**

### 1. في الشريط العلوي ستجد أيقونة "⋯" (ثلاث نقاط)
**اضغط عليها**

### 2. ستظهر قائمة منسدلة، اختر:
```
Import from JSON
```

### 3. ستفتح نافذة صغيرة بها مربع نص كبير

### 4. الآن افتح ملف `IMPORT_WORKFLOWS_GUIDE.md` واذهب لقسم:
```
🚀 Workflow الأول: Sell Process Started
```

### 5. انسخ كل النص الموجود داخل:
```json
{
  "name": "Globul Cars - Sell Process Started",
  ... (كل المحتوى حتى آخر قوس)
}
```

### 6. ارجع لنافذة n8n والصق النص في المربع

### 7. اضغط زر **"Import"**

---

## 💾 **الخطوة الرابعة: حفظ وتفعيل**

### 1. ستظهر لك الـ workflow بشكل مرئي (صناديق مربوطة ببعض)

### 2. في الشريط العلوي اضغط زر **"Save"**
- سيُطلب منك اسم، اتركه كما هو أو غيّره

### 3. بعد الحفظ، اضغط زر **"Active"** 
- سيتحول لون الزر للأخضر
- هذا يعني أن الـ workflow يعمل الآن

---

## 🔄 **كرر نفس العملية للـ Workflow الثاني**

### 1. ارجع للصفحة الرئيسية (اضغط شعار n8n في الأعلى)

### 2. اضغط **"+ New Workflow"** مرة أخرى

### 3. اضغط "⋯" → **"Import from JSON"**

### 4. هذه المرة افتح ملف `WORKFLOW_2_JSON.md`

### 5. انسخ النص من:
```json
{
  "name": "Globul Cars - Vehicle Type Selected",
  ... (كل المحتوى حتى آخر قوس)
}
```

### 6. الصق في n8n واضغط **"Import"**

### 7. **"Save"** ثم **"Active"**

---

## ✅ **التأكد من النجاح**

### بعد الانتهاء ستجد في حسابك:

1. **Workflow الأول**: "Globul Cars - Sell Process Started"
   - حالته: Active (أخضر)

2. **Workflow الثاني**: "Globul Cars - Vehicle Type Selected"  
   - حالته: Active (أخضر)

---

## 🧪 **اختبار سريع**

### 1. افتح أي من الـ workflows

### 2. اضغط على الصندوق الأول (Webhook)

### 3. ستجد في الجانب رابط مثل:
```
https://globul-cars-bg.app.n8n.cloud/webhook/sell-started
```
**هذا هو الرابط الذي سنستخدمه للاختبار**

---

## 🤔 **إذا واجهت مشاكل**

### مشكلة: لم أجد زر "Import from JSON"
**الحل**: تأكد أنك في صفحة محرر الـ workflow (ليس الصفحة الرئيسية)

### مشكلة: خطأ عند الاستيراد
**الحل**: تأكد من نسخ JSON كاملاً من أول `{` إلى آخر `}`

### مشكلة: لا يمكنني التفعيل
**الحل**: احفظ أولاً بـ "Save" ثم اضغط "Active"
