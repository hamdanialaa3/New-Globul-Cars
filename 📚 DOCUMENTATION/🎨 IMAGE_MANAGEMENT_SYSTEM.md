# 🎨 نظام إدارة الصور في صفحة تفاصيل السيارة

## 📋 **نظرة عامة**

تم تطوير نظام كامل لإدارة صور السيارات في صفحة التفاصيل (`CarDetailsPage.tsx`)، يسمح لمالك السيارة بإضافة وحذف الصور مباشرة من صفحة التفاصيل في وضع التعديل.

---

## ✨ **الميزات الرئيسية**

### **1. رفع صور جديدة** 📤
- **الموقع:** Panel في أعلى قسم الصور (يظهر فقط في وضع التعديل)
- **الحد الأقصى:** 20 صورة للسيارة الواحدة
- **الحجم الأقصى:** 5MB لكل صورة
- **الصيغ المدعومة:** جميع صيغ الصور (jpg, png, webp, gif, etc.)
- **رفع متعدد:** يمكن اختيار عدة صور دفعة واحدة

### **2. حذف صور موجودة** 🗑️
- **الموقع:** زر × أحمر في الزاوية العلوية اليمنى لكل صورة
- **التأكيد:** يظهر نافذة تأكيد قبل الحذف
- **المعاينة:** الصور المحددة للحذف تختفي فوراً من المعاينة
- **الإلغاء:** يمكن إلغاء وضع التعديل لاستعادة الصور المحذوفة

### **3. معاينة الصور الجديدة** 👁️
- **عرض مباشر:** الصور المضافة تظهر فوراً في panel خاص
- **إطار أخضر:** تمييز الصور الجديدة بإطار أخضر
- **زر إزالة أخضر:** يمكن إزالة الصور الجديدة قبل الحفظ

### **4. عداد الصور الديناميكي** 📊
```
Снимки: 8 / 20  (البلغارية)
Images: 8 / 20  (الإنجليزية)
```
يحدث العداد تلقائياً حسب:
- الصور الموجودة
- الصور المحذوفة
- الصور الجديدة المضافة

## 📝 **سير العمل (Workflow)**

### **1. الدخول لوضع التعديل:**
```
المستخدم → زر "Редактирай" → isEditMode = true
```

### **2. إضافة صور:**
```
1. النقر على "Добави снимки"
2. اختيار الصور
3. التحقق من النوع والحجم
4. إضافة للـ newImages state
5. عرض معاينة فورية
```

### **3. حذف صور:**
```
1. النقر على زر × الأحمر
2. تأكيد الحذف
3. إضافة للـ imagesToDelete state
4. إخفاء من المعاينة
```

### **4. الحفظ:**
```
1. رفع الصور الجديدة → Firebase Storage
2. الحصول على الروابط
3. حذف الصور المحددة من Storage
4. تحديث مصفوفة images في Firestore
5. حفظ جميع التغييرات
6. الخروج من وضع التعديل
```

## 🚀 **الاستخدام**

### **للمستخدم:**
1. افتح صفحة السيارة: `http://localhost:3000/car/{carId}`
2. اضغط على زر **"Редактирай"** (أعلى اليمين)
3. اضغط على **"Добави снимки"** لإضافة صور
4. اضغط على **×** الأحمر لحذف صور موجودة
5. اضغط على **"Запази"** لحفظ التغييرات

### **للمطور:**
```typescript
// إضافة صور جديدة
const uploadedUrls = await carListingService.uploadImages(carId, files);

// حذف صور
await carListingService.deleteImages(carId, imageUrls);

// تحديث السيارة
await carListingService.updateListing(carId, { images: updatedImages });
```
