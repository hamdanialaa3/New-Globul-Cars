#### Pattern للإصلاح:

1. **أضف Import في بداية الملف:**
```typescript
import { serviceLogger } from './logger-wrapper';
```

2. **استبدل console.error:**
```typescript
// ❌ قبل:
console.error('Error getting data:', error);

// ✅ بعد:
serviceLogger.error('Error getting data', error as Error, { userId, operation: 'getData' });
```

3. **استبدل console.log:**
```typescript
// ❌ قبل:
console.log('✅ Operation completed');

// ✅ بعد:
serviceLogger.info('Operation completed', { userId, timestamp: Date.now() });
```

4. **استبدل console.warn:**
```typescript
// ❌ قبل:
console.warn('Deprecated method');

// ✅ بعد:
serviceLogger.warn('Deprecated method', { method: 'oldMethod', replacement: 'newMethod' });
```

## 💡 نصائح مهمة

### 1. Context is King
```typescript
// 🔴 ضعيف:
serviceLogger.error('Error', error as Error);

// ✅ ممتاز - أضف context مفيد:
serviceLogger.error('Error getting user data', error as Error, { 
  userId: 'user_123',
  operation: 'getUserData',
  timestamp: Date.now()
});
```

### 2. استخدم VS Code Find & Replace
```
في VS Code:
Ctrl+H (Find & Replace)

Find:    console\.error\('([^']+)',\s*error\);?
Replace: serviceLogger.error('$1', error as Error);

⚠️ لكن دائماً راجع يدوياً وأضف context!
```

### 3. تحقق من TypeScript Errors
بعد كل تعديل:
```
File → Problems (أو Ctrl+Shift+M)
```

تأكد من عدم وجود أخطاء compilation.
