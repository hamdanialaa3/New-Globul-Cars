## 🚗 نظام بيع السيارات (Sell Workflow)

### 📋 الخطوات (Mobile.de Style)

```
1. اختيار نوع المركبة
   /sell/auto

2. نوع البائع (فرد/تاجر)
   /sell/inserat/:vehicleType/verkaeufertyp

3. بيانات المركبة الأساسية
   /sell/inserat/:vehicleType/fahrzeugdaten/antrieb-und-umwelt

4. التجهيزات (صفحة موحدة)
   /sell/inserat/:vehicleType/equipment

5. رفع الصور
   /sell/inserat/:vehicleType/details/bilder

6. تحديد السعر
   /sell/inserat/:vehicleType/details/preis

7. بيانات الاتصال (صفحة موحدة)
   /sell/inserat/:vehicleType/contact

8. المعاينة
   /sell/inserat/:vehicleType/preview

9. الإرسال النهائي
   /sell/inserat/:vehicleType/submission
```

### 💾 حفظ البيانات

- **localStorage:** حفظ تلقائي في كل خطوة
- **Firestore Drafts:** حفظ المسودات في قاعدة البيانات
- **WorkflowPersistenceService:** خدمة موحدة للحفظ والاسترجاع

### 🔄 تدفق البيانات

```
User Input
    ↓
WorkflowPersistenceService.saveState()
    ↓
localStorage + Firestore Drafts
    ↓
عند الإرسال النهائي:
    ↓
SellWorkflowService.createCarListing()
    ↓
Firestore Collection: 'cars'
    ↓
Algolia Index (للبحث)
```
