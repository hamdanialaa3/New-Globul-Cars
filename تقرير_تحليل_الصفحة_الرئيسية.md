## 3. قسم Hero - البحث الرئيسي

### 3.1 المكون الرئيسي
**UnifiedHeroSection** - يحتوي على **SearchWidget**

### 3.2 التصميم
- **الخلفية:** Gradient ديناميكي (Dark/Light حسب المظهر)
- **الحاوية:** Container بعرض 1400px كحد أقصى
- **المحتوى:** متمركز مع SearchWidget

### 3.3 SearchWidget - ويدجت البحث

#### أ. التبويبات (Tabs)
ثلاثة تبويبات:
1. **🚗 All Cars (Всички)** - جميع السيارات
   - التبويب النشط الافتراضي
2. **✅ Used (Употребявани)** - مستعملة
   - ينقل إلى `/sell/auto?condition=used`
3. **🆕 New (Нови)** - جديدة
   - ينقل إلى `/sell/auto?condition=new`

#### ب. حقول البحث (Form Fields)

**الصف الأول:**
1. **Make (Марка) - العلامة التجارية**
   - نوع: Dropdown Select
   - الخيارات: جميع العلامات التجارية المتاحة
   - الخيار الافتراضي: "Any Make (Всички марки)"
   - يتم تحميل القائمة من `brandsModelsDataService`

2. **Model (Модел) - الموديل**
   - نوع: Dropdown Select
   - الخيارات: تعتمد على العلامة التجارية المختارة (Cascading)
   - الخيار الافتراضي: "Any Model (Всички модели)"
   - **معطل** إذا لم يتم اختيار Make
   - يتم تحميل القائمة ديناميكياً عند تغيير Make

3. **First Reg. (Година от) - سنة التسجيل**
   - نوع: Dropdown Select
   - الخيارات: آخر 30 سنة (من السنة الحالية إلى 1995)
   - الخيار الافتراضي: "Any Year (От година)"

4. **Price up to (Цена до) - السعر حتى**
   - نوع: Dropdown Select
   - الخيارات:
     - € 5,000
     - € 10,000
     - € 15,000
     - € 20,000
     - € 30,000
     - € 50,000
     - € 75,000
     - € 100,000
   - الخيار الافتراضي: "Any Price (До цена)"

#### ج. الأزرار

**1. زر البحث الرئيسي (Search Button)**
- **النص:** "Show [رقم عشوائي] offers" / "Покажи [رقم] обяви"
- **الأيقونة:** 🔍 Search icon
- **الوظيفة:** ينقل إلى `/advanced-search` مع المعاملات المختارة
- **التصميم:** Orange gradient button
- **الموقع:** بجانب حقول البحث

**2. رابط البحث المتقدم (Advanced Search Link)**
- **النص:** "Detailed Search (Подробно търсене)"
- **الموقع:** أسفل حقول البحث
- **الوظيفة:** ينقل إلى `/advanced-search`

### 3.4 الميزات الإضافية
- **Responsive Design:** تصميم متجاوب كامل
- **Dark Mode Support:** دعم الوضع الداكن
- **Input Sanitization:** تنظيف المدخلات (sanitizeCarMakeModel)
- **Loading States:** حالات تحميل للعلامات التجارية والموديلات
