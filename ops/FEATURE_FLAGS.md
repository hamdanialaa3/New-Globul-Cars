Escalation\n\nIf a flagged feature is causing production issues:\n\n1. **Immediate**: Set flag to `false` in Firebase Remote Config\n2. **Notify**: Slack #incidents + on-call engineer\n3. **Assess**: Determine root cause\n4. **Rollback**: Deploy fix and re-enable or remove feature\n\nEstimated resolution time: **5 minutes** (no deploy needed)\n\n## How to Add a New Flag\n\n1. Define in Firebase Remote Config console\n2. Add entry to table above\n3. Integrate into code:\n\n```typescript\nimport { getRemoteConfig, getValue } from 'firebase/remote-config';\n\nconst remoteConfig = getRemoteConfig();\nconst isFeatureEnabled = getValue(remoteConfig, 'FEATURE_NAME').asBoolean();\n\nif (isFeatureEnabled) {\n  // Show feature\n}\n```\n\n4. Set up analytics tracking: `logEvent('feature_flag_used', { flag: 'FEATURE_NAME', enabled: true })`\n5. Test in staging with flag toggled off/on\n6. Enable for canary rollout: 10% → 50% → 100%\n\n## Rollout Strategy\n\n### Gradual Rollout (A/B Testing)\n```\nDay 1:  10% of users\nDay 2:  25% of users (if no errors)\nDay 3:  50% of users\nDay 5:  100% of users\n```\n\n### Monitoring During Rollout\n- Error rate in Sentry\n- Conversion rate in GA\n- User feedback in support tickets\n- Server logs (latency, queue depth)\n\n### Rollback Criteria\n- Error rate spikes > 2% above baseline\n- Conversion drops > 5%\n- User complaints in support\n- Performance regression (p95 latency > 500ms)\n\n## Code Examples\n\n### React Component with Flag\n```typescript\nimport { useRemoteConfig } from '../hooks/useRemoteConfig';\n\nexport const ListingWizard = () => {\n  const { isFeatureEnabled } = useRemoteConfig();\n  const showAISuggestions = isFeatureEnabled('AI_VISION_SUGGEST');\n\n  return (\n    <div>\n      {showAISuggestions && <AISuggestionsPanel />}\n      {/* rest of wizard */}\n    </div>\n  );\n};\n```\n\n### Server-Side Flag Check\n```typescript\nexport const createListing = async (req, res) => {\n  const slugEnabled = await remoteConfig.getBoolean('ENABLE_SLUG_SERVICE');\n\n  if (slugEnabled) {\n    const slug = await SlugService.generateUniqueSlug(listingData);\n    listing.slug = slug;\n  }\n\n  await saveListing(listing);\n  res.status(201).send({ listingId: listing.id });\n};\n```\n\n## Sunset Policy\n\nFlags should be cleaned up after 90 days if fully rolled out:\n\n1. Remove flag logic from code\n2. Delete from Firebase Remote Config\n3. Update this documentation\n4.
