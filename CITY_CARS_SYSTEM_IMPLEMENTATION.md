### الإحداثيات الجغرافية
```
🌍 حدود بلغاريا:
├── Latitude:  41.2°N - 44.2°N
├── Longitude: 22.4°E - 28.6°E
└── دقة: 4 منازل عشرية
```

### التحويل إلى SVG
```typescript
const coordToSVG = (lat: number, lng: number) => {
  const x = ((lng - 22.4) / (28.6 - 22.4)) * 800;
  const y = ((44.2 - lat) / (44.2 - 41.2)) * 450;
  return { x, y };
};
```
