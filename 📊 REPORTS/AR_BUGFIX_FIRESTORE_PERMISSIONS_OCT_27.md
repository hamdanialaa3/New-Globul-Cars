# إصلاح خطأ الصلاحيات - إضافة السيارات
**التاريخ:** 27 أكتوبر 2025  
**الحالة:** ✅ تم الإصلاح

---

## المشكلة

### الخطأ
```
Missing or insufficient permissions.
```

### الموقع
عند محاولة نشر إعلان سيارة في الخطوة الأخيرة من نموذج البيع.

---

## السبب

قواعد Firestore كانت تتطلب أن يكون لدى المستخدم **Custom Claim** اسمه `seller: true` لكي يتمكن من إنشاء إعلان.

**القاعدة القديمة:**
```javascript
// يسمح فقط للبائعين والمسؤولين
allow create: if (isSeller() || isAdmin()) && ...
```

**المشكلة:**
- المستخدمون العاديون ليس لديهم `seller: true` في الـ token
- لا توجد آلية لإعطاء المستخدمين هذه الصلاحية تلقائياً
- النتيجة: **رفض الإذن** ❌

---

## الحل

### تعديل القاعدة

**القاعدة الجديدة:**
```javascript
// يسمح لأي مستخدم مُصادق عليه
allow create: if isSignedIn() && 
                 request.resource.data.sellerId == request.auth.uid &&
                 // ... باقي التحققات
```

### ما تم تغييره

**قبل:**
```
✅ مستخدم مُصادق؟
✅ معلومات صحيحة؟
❌ Custom Claim موجود؟ ← هنا كان الخطأ
```

**بعد:**
```

✅ معلومات صحيحة؟

```

القاعدة الجديدة لا تزال تتحقق من:

- ✅ المستخدم مُصادق عليه
- ✅ الإعلان مملوك للمستخدم نفسه
- ✅ البريد الإلكتروني متطابق
- ✅ جميع الحقول المطلوبة موجودة
- ✅ السعر صحيح (> 0)
