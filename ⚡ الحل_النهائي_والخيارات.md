## 🔍 **شرح الخطأ (للمعرفة):**

### **ما يحدث خطوة بخطوة:**

```javascript
// 1. الكود القديم في localhost cache:
where('status', '!=', 'read')

// 2. Firestore يجد document:
{ status: null }

// 3. Firestore internally يحاول:
if ('nullValue' in null)  ← CRASH!

// 4. Error:
Cannot use 'in' operator to search for 'nullValue' in null

// 5. الحل (في الكود الجديد):
// Client-side filter بدلاً من server-side
.filter(doc => doc.data().status !== 'read')
```
