## 🚀 خطوات النشر اليدوية

إذا لم تعمل السكريبتات، يمكنك تنفيذ الأوامر يدوياً:

### 1. إضافة جميع التغييرات إلى Git
```bash
cd "c:\Users\hamda\Desktop\New Globul Cars"
git add -A
```

### 2. عمل Commit
```bash
git commit -m "docs: توحيد وتنظيف الملفات التوثيقية - دمج 16 ملف في ملفين موحدين"
```

### 3. رفع إلى GitHub
```bash
git push origin main
```

### 4. بناء المشروع
```bash
cd bulgarian-car-marketplace
npm run build
```

### 5. النشر إلى Firebase
```bash
cd ..
firebase deploy --only hosting
```
