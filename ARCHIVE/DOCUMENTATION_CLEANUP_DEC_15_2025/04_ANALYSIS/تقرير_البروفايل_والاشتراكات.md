## 🎯 نظرة عامة

نظام البروفايل والاشتراكات في Globul Cars هو نظام متكامل يدير **3 أنواع من البروفايلات** و **3 خطط اشتراك** مع تكامل كامل مع Stripe للدفع.

### المكونات الرئيسية:
- ✅ **3 أنواع بروفايل:** Private (خاص), Dealer (تاجر), Company (شركة)
- ✅ **3 خطط اشتراك:** Free, Dealer, Company
- ✅ **نظام أذونات ديناميكي** يعتمد على النوع والخطة
- ✅ **تكامل Stripe** للدفع والاشتراكات
- ✅ **Real-time synchronization** مع Firestore
- ✅ **Theme system** ديناميكي (ألوان مختلفة لكل نوع)

## 🏗️ البنية المعمارية

### 1. Context Layer (طبقة السياق)

**الملف الرئيسي:** `src/contexts/ProfileTypeContext.tsx`

#### الوظائف:
- إدارة حالة `profileType` و `planTier` عالمياً
- Real-time listener على `users/{uid}`
- حساب الأذونات ديناميكياً
- إدارة Theme colors
- التحقق من صحة التبديل بين الأنواع

#### الـ State:
```typescript
interface ProfileTypeContextState {
  profileType: ProfileType;  // 'private' | 'dealer' | 'company'
  theme: ProfileTheme;        // الألوان الديناميكية
  permissions: ProfilePermissions;  // الأذونات المحسوبة
  planTier: PlanTier;         // 'free' | 'dealer' | 'company'
  loading: boolean;
  
  // Helper booleans
  isPrivate: boolean;
  isDealer: boolean;
  isCompany: boolean;
  
  // Actions
  switchProfileType: (newType: ProfileType) => Promise<void>;
  refreshProfileType: () => Promise<void>;
}
```

#### الاستخدام:
```typescript
const { profileType, planTier, permissions, theme } = useProfileType();
```

```typescript
function getPermissions(profileType: ProfileType, planTier: PlanTier): ProfilePermissions {
  // Base permissions for all
  const base: ProfilePermissions = {
    canAddListings: true,
    maxListings: PLAN_LIMITS[planTier] || 5,
    hasAnalytics: false,
    hasAdvancedAnalytics: false,
    hasTeam: false,
    canExportData: false,
    hasPrioritySupport: false,
    canUseQuickReplies: false,
    canBulkEdit: false,
    canImportCSV: false,
    canUseAPI: false
  };

  // Dealer enhancements
  if (profileType === 'dealer') {
    return {
      ...base,
      hasAnalytics: true,
      canUseQuickReplies: true,
      canBulkEdit: true,
      // Advanced features for dealer_pro/enterprise
      hasAdvancedAnalytics: planTier === 'dealer_pro' || planTier === 'dealer_enterprise',
      canExportData: planTier === 'dealer_pro' || planTier === 'dealer_enterprise',
      canImportCSV: planTier === 'dealer_pro' || planTier === 'dealer_enterprise',
      canUseAPI: planTier === 'dealer_enterprise'
    };
  }

  // Company enhancements
  if (profileType === 'company') {
    return {
      ...base,
      hasAnalytics: true,
      hasAdvancedAnalytics: true,
      hasTeam: true,
      canExportData: true,
      canUseQuickReplies: true,
      canBulkEdit: true,
      canImportCSV: planTier !== 'company_starter',
      canUseAPI: planTier === 'company_pro' || planTier === 'company_enterprise',
      hasPrioritySupport: true
    };
  }

  return base;
}
```
