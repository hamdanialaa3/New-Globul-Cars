## 🔘 Toggle Switch Component (Neumorphism)

### الوصف
مكون تبديل (Toggle Switch) بتصميم Neumorphism مع تأثير النيون الأخضر عند التفعيل. يستخدم في عناصر Privacy Settings مثل:
- Show Phone
- Show Email
- Show Last Seen
- Allow Messages
- Show Activity

### البنية والتصميم

#### 1. **SwitchRow** (المستطيل الخارجي)
```typescript
const SwitchRow = styled.div<{ $active: boolean }>`
  display: flex;
  align-items: center;
  gap: 16px;
  width: 100%;
  padding: 16px 20px;
  background: rgba(255, 255, 255, 0.03);
  border-radius: 12px;
  cursor: pointer;
  transition: all 0.3s ease;
  border: 2px solid ${props => props.$active ? 'rgba(15, 255, 0, 0.3)' : 'transparent'};
```
**الخصائص:**
- **الخلفية**: شفافة بيضاء (3% opacity)
- **الحدود**: تظهر باللون الأخضر الشفاف (30% opacity) عند التفعيل
- **التأثير عند Hover**: يتحرك للأعلى قليلاً (`translateY(-2px)`) وتزداد شفافية الخلفية

#### 2. **SwitchWrapper** (حاوية الزر)
```typescript
const SwitchWrapper = styled.div<{ $active: boolean }>`
  position: relative;
  width: 50px;
  height: 25px;
  background: #3e3e3e;
  border-radius: 12.5px;
  box-shadow: 
    5px 5px 10px rgba(0, 0, 0, 0.4), 
    -5px -5px 10px rgba(255, 255, 255, 0.1);
```
**الخصائص:**
- **الحجم**: 50px × 25px (نصف الحجم الأصلي)
- **الخلفية**: رمادي داكن (#3e3e3e)
- **الظل**: Neumorphism effect (ظل خارجي + ظل داخلي)

#### 3. **SwitchInner** (الطبقة الداخلية)
```typescript
const SwitchInner = styled.div<{ $active: boolean }>`
  position: absolute;
  top: 2.5px;
  left: 2.5px;
  width: calc(100% - 5px);
  height: calc(100% - 5px);
  background-color: #3e3e3e;
  border-radius: 10px;
  box-shadow: 
    inset 2.5px 2.5px 5px rgba(0, 0, 0, 0.4), 
    inset -2.5px -2.5px 5px rgba(255, 255, 255, 0.1);
```
**الخصائص:**
- **الظل الداخلي**: يخلق تأثير Neumorphism ثلاثي الأبعاد
- **الموضع**: داخل SwitchWrapper مع padding 2.5px من جميع الجهات

#### 4. **SwitchKnobContainer** (حاوية المقبض المتحرك)
```typescript
const SwitchKnobContainer = styled.div<{ $active: boolean }>`
  position: absolute;
  top: 0;
  left: 0;
  width: 50%;
  height: 100%;
  overflow: hidden;
  transition: transform 0.4s ease;
  transform: ${props => props.$active ? 'translateX(100%)' : 'translateX(0)'};
```
**الآلية:**
- **الحالة غير المفعلة**: `translateX(0)` - المقبض في اليسار
- **الحالة المفعلة**: `translateX(100%)` - المقبض ينتقل لليمين
- **الحركة**: انتقال سلس لمدة 0.4 ثانية

#### 5. **SwitchKnob** (المقبض الدائري)
```typescript
const SwitchKnob = styled.div<{ $active: boolean }>`
  position: relative;
  width: 20px;
  height: 20px;
  top: 2.5px;
  left: 2.5px;
  background-color: #3e3e3e;
  border-radius: 50%;
  box-shadow: 
    2.5px 2.5px 5px rgba(0, 0, 0, 0.5), 
    -2.5px -2.5px 5px rgba(255, 255, 255, 0.1);
```
**الخصائص:**
- **الشكل**: دائري تماماً (`border-radius: 50%`)
- **الحجم**: 20px × 20px
- **الظل**: Neumorphism effect للمقبض

#### 6. **SwitchKnobNeon** (تأثير النيون)
```typescript
const SwitchKnobNeon = styled.div<{ $active: boolean }>`
  position: absolute;
  top: 50%;
  left: 50%;
  width: 12.5px;
  height: 12.5px;
  border-radius: 50%;
  transform: translate(-50%, -50%);
  box-shadow: ${props => props.$active 
    ? '0 0 5px #0f0, 0 0 10px #0f0, 0 0 15px #0f0, 0 0 20px #0f0'
    : '0 0 5px #ff8c00, 0 0 10px #ff8c00'};
```
**الآلية:**
- **عند التفعيل**: تأثير نيون أخضر (#0f0) مع 4 طبقات من التوهج
- **عند الإيقاف**: تأثير نيون برتقالي (#ff8c00) مع طبقتين من التوهج
- **الموضع**: في مركز المقبض تماماً

### كيفية الاستخدام

```tsx
<ToggleRow
  $active={settings.privacy.showPhone}
  onClick={() => setSettings({
    ...settings,
    privacy: { ...settings.privacy, showPhone: !settings.privacy.showPhone }
  })}
>
  <ToggleLabel>
    <Phone size={18} />
    <div>
      <strong>{t('settings.showPhone', 'Show Phone Number')}</strong>
      <HelpText>{t('settings.showPhoneHelp', 'Visible on your listings')}</HelpText>
    </div>
  </ToggleLabel>
</ToggleRow>
```

### المكونات المطلوبة

1. **ToggleRow**: الصف الرئيسي الذي يحتوي على التسمية والزر
2. **ToggleLabel**: يحتوي على الأيقونة والنص
3. **SwitchWrapper**: حاوية الزر (يتم إضافتها تلقائياً داخل ToggleRow)

### الألوان المستخدمة

- **الخلفية**: `#3e3e3e` (رمادي داكن)
- **النيون الأخضر (مفعل)**: `#0f0`
- **النيون البرتقالي (غير مفعل)**: `#ff8c00`
- **الحدود الخضراء**: `rgba(15, 255, 0, 0.3)`

### الحركات والانتقالات

- **مدة الانتقال**: 0.4 ثانية (`transition: all 0.4s ease`)
- **نوع الانتقال**: `ease` (سلس)
- **الحركة**: المقبض ينتقل من اليسار إلى اليمين عند التفعيل

## 🔵 Animated Button Component

### الوصف
زر متحرك مع حدود متحركة باللون الأزرق. يستخدم في أزرار الحفظ (Save) وأزرار أخرى في التطبيق.

### البنية والتصميم

#### 1. **AnimatedButtonBase** (الزر الأساسي)
```typescript
const AnimatedButtonBase = styled.button<{ $color?: string }>`
  position: relative;
  display: flex;
  align-items: center;
  justify-content: center;
  gap: 8px;
  padding: 14px 24px;
  color: ${props => props.$color || '#1670f0'};
  text-transform: uppercase;
  text-decoration: none;
  letter-spacing: 2px;
  font-size: 0.95rem;
  font-weight: 600;
  cursor: pointer;
  overflow: hidden;
  box-shadow: 0 20px 50px rgba(0, 0, 0, 0.3);
  border: none;
  background: transparent;
  transition: all 0.3s ease;
```
**الخصائص:**
- **اللون الافتراضي**: أزرق (#1670f0)
- **النص**: أحرف كبيرة (`uppercase`) مع مسافة بين الأحرف (`letter-spacing: 2px`)
- **الخلفية**: شفافة (`transparent`)
- **الظل**: ظل قوي للعمق (`0 20px 50px rgba(0, 0, 0, 0.3)`)

#### 2. **الطبقة الداخلية (::before)**
```typescript
&::before {
  content: "";
  position: absolute;
  top: 2px;
  left: 2px;
  bottom: 2px;
  width: 50%;
  background: rgba(255, 255, 255, 0.05);
}
```
**الوظيفة**: طبقة شفافة في النصف الأيسر من الزر لإضافة عمق بصري

#### 3. **الحدود المتحركة (4 خطوط)**

##### الخط العلوي (من اليسار إلى اليمين)
```typescript
> span.btn-span:nth-of-type(1) {
  top: 0;
  left: 0;
  width: 100%;
  height: 2px;
  background: linear-gradient(to right, #0c002b, ${props => props.$color || '#1670f0'});
  animation: animate1 2s linear infinite;
  animation-delay: 1s;
}

@keyframes animate1 {
  0% { transform: translateX(-100%); }
  100% { transform: translateX(100%); }
}
```
**الحركة**: يبدأ من خارج اليسار (`-100%`) وينتقل للخارج من اليمين (`100%`)
**التأخير**: 1 ثانية قبل البدء
**المدة**: 2 ثانية لكل دورة

##### الخط الأيمن (من الأعلى إلى الأسفل)
```typescript
> span.btn-span:nth-of-type(2) {
  top: 0;
  right: 0;
  width: 2px;
  height: 100%;
  background: linear-gradient(to bottom, #0c002b, ${props => props.$color || '#1670f0'});
  animation: animate2 2s linear infinite;
  animation-delay: 2s;
}

@keyframes animate2 {
  0% { transform: translateY(-100%); }
  100% { transform: translateY(100%); }
}
```
**الحركة**: يبدأ من خارج الأعلى وينتقل للأسفل
**التأخير**: 2 ثانية

##### الخط السفلي (من اليمين إلى اليسار)
```typescript
> span.btn-span:nth-of-type(3) {
  bottom: 0;
  left: 0;
  width: 100%;
  height: 2px;
  background: linear-gradient(to left, #0c002b, ${props => props.$color || '#1670f0'});
  animation: animate3 2s linear infinite;
  animation-delay: 1s;
}

@keyframes animate3 {
  0% { transform: translateX(100%); }
  100% { transform: translateX(-100%); }
}
```
**الحركة**: يبدأ من خارج اليمين وينتقل للخارج من اليسار
**التأخير**: 1 ثانية

##### الخط الأيسر (من الأسفل إلى الأعلى)
```typescript
> span.btn-span:nth-of-type(4) {
  top: 0;
  left: 0;
  width: 2px;
  height: 100%;
  background: linear-gradient(to top, #0c002b, ${props => props.$color || '#1670f0'});
  animation: animate4 2s linear infinite;
  animation-delay: 2s;
}

@keyframes animate4 {
  0% { transform: translateY(100%); }
  100% { transform: translateY(-100%); }
}
```
**الحركة**: يبدأ من خارج الأسفل وينتقل للأعلى
**التأخير**: 2 ثانية

#### 4. **AnimatedButton Component** (الغلاف التلقائي)
```typescript
const AnimatedButton: React.FC<React.ButtonHTMLAttributes<HTMLButtonElement> & { $color?: string }> = ({ children, $color, ...props }) => {
  return (
    <AnimatedButtonBase $color={$color} {...props}>
      <span className="btn-span"></span>
      <span className="btn-span"></span>
      <span className="btn-span"></span>
      <span className="btn-span"></span>
      {children}
    </AnimatedButtonBase>
  );
};
```
**الوظيفة**: يضيف تلقائياً 4 عناصر `<span>` للحدود المتحركة

#### 5. **SaveButton** (مثال للاستخدام)
```typescript
const SaveButton = (props: React.ButtonHTMLAttributes<HTMLButtonElement>) => (
  <AnimatedButton $color="#1670f0" {...props} />
);
```
**اللون**: أزرق (#1670f0)

### كيفية الاستخدام

```tsx
<SaveButton onClick={handleSave} disabled={saving}>
  {saving ? (
    <>
      <Spinner />
      {t('common.saving', 'Saving...')}
    </>
  ) : (
    <>
      <Save size={18} />
      {t('common.save', 'Save')}
    </>
  )}
</SaveButton>
```

### المكونات المطلوبة

1. **AnimatedButtonBase**: الزر الأساسي مع الأنماط
2. **AnimatedButton**: الغلاف الذي يضيف الحدود المتحركة تلقائياً
3. **4 عناصر span**: للحدود المتحركة (يتم إضافتها تلقائياً)

### الألوان المستخدمة

- **اللون الأساسي**: `#1670f0` (أزرق)
- **اللون الداكن**: `#0c002b` (أسود مزرق للـ gradient)
- **اللون المخصص**: يمكن تمرير أي لون عبر prop `$color`

### الحركات والانتقالات

- **مدة كل حركة**: 2 ثانية
- **نوع الحركة**: `linear infinite` (مستمرة بلا نهاية)
- **التأخير**: 
  - الخطوط 1 و 3: 1 ثانية
  - الخطوط 2 و 4: 2 ثانية
- **الاتجاه**: حركة دائرية حول الزر (علوي → يمين → سفلي → يسار)

### تأثير Hover

```typescript
&:hover:not(:disabled) {
  transform: translateY(-2px);
  box-shadow: 0 25px 60px rgba(0, 0, 0, 0.4);
}
```
**الحركة**: يتحرك للأعلى قليلاً عند التمرير
**الظل**: يزداد قوة الظل

### حالة Disabled

```typescript
&:disabled {
  opacity: 0.5;
  cursor: not-allowed;
  
  > span.btn-span {
    animation: none;
  }
}
```
**السلوك**: عند تعطيل الزر، تتوقف جميع الحركات وتقل الشفافية

## 📝 ملاحظات مهمة

### Toggle Switch
- الحجم الحالي هو **نصف الحجم الأصلي** (50px × 25px)
- التأثير النيون الأخضر يظهر فقط عند التفعيل
- الحدود الخضراء تظهر فقط عند التفعيل

### Animated Button
- الحدود المتحركة تعمل بشكل مستمر (infinite)
- يمكن تخصيص اللون عبر prop `$color`
- الحركة دائرية وتتكرر كل 2 ثانية

## 🔄 إعادة الاستخدام

### استخدام Toggle Switch في أماكن أخرى

```tsx
import { ToggleRow, ToggleLabel } from './SettingsTab';

<ToggleRow
  $active={someState}
  onClick={() => setSomeState(!someState)}
>
  <ToggleLabel>
    <SomeIcon size={18} />
    <div>
      <strong>عنوان الخيار</strong>
      <HelpText>وصف الخيار</HelpText>
    </div>
  </ToggleLabel>
</ToggleRow>
```

### استخدام Animated Button في أماكن أخرى

```tsx
import { AnimatedButton } from './SettingsTab';

<AnimatedButton $color="#10b981" onClick={handleAction}>
  <Save size={18} />
  حفظ
</AnimatedButton>
```

**ألوان مقترحة:**
- أزرق: `#1670f0` (Save)
- أخضر: `#10b981` (Success)
- أحمر: `#ef4444` (Danger)
- رمادي: `#6b7280` (Cancel)
