## 🖱️ **تأثيرات Hover**

### عند المرور على الـ Navigation Bar:

```
قبل hover:
🧡━━━━━━━━━━━━━━━━━━━━━━━━━━━⬛
   (لمعان عادي)

أثناء hover:
🧡━━━━━━━━✨━━━━━━━━━━━━━━━━⬛
   (شعاع ضوئي يمر بسرعة)
          ↓
     يمر في 0.8 ثانية
```

### عند المرور على رابط:

```
قبل:
[ Home ]  [ Cars ]  [ Sell ]
  أبيض     أبيض     أبيض

hover على Cars:
[ Home ]  [ Cars ]  [ Sell ]
           ↓
        ┌───────┐
        │ Cars  │  ← ذهبي + يرتفع
        └───────┘
           ━━━     ← خط ذهبي يظهر
           ✨      ← توهج
```

## 🎬 **السيناريوهات**

### سيناريو 1: المستخدم يفتح الموقع

```
1. الصفحة تُحمل
2. Header يظهر
3. Navigation bar بتدريج برتقالي → أسود ✨
4. لمعان معدني خفيف يتحرك ببطء
5. نبضات خفيفة كل 4 ثوانٍ
```

### سيناريو 2: المستخدم يمرر الماوس

```
1. Mouse يقترب من Navigation
2. شعاع ضوئي يمر بسرعة ✨
3. المستخدم يمرر على "Cars"
4. الرابط:
   - يتحول للذهبي
   - يرتفع 2px
   - خط ذهبي يظهر
   - توهج حوله
```

### سيناريو 3: المستخدم في صفحة معينة

```
1. المستخدم في صفحة Cars
2. رابط "Cars" نشط:
   - لون ذهبي
   - خط ذهبي دائم
   - توهج خفيف
   - يبرز عن البقية
```
